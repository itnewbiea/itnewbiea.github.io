MySQL之将查询结果生成新表

SQL语句示例：

CREATE TABLE school SELECT * FROM class

CREARE负责创建表，表的内容是SELECT语句的查询结果

 

如果我们想结合两个表的相同字段并生成新表，SQL语句如下：

CREATE TABLE school SELECT * FROME CLASS1,CLASS2 WHERE CLASS1.NAME=CLASS2.NAME
