安装docker-compose

一、在线安装（需要梯子）

[root@lgb /]# curl -L https://github.com/docker/compose/releases/download/1.21.0/docker-compose-$(uname -s)-$(uname -m) -o /usr/local/bin/docker-compose

[root@lgb bin]# chmod +x /usr/local/bin/docker-compose

[root@lgb bin]# docker-compose -version
docker-compose version 1.29.0, build 07737305

二、离线安装

https://github.com/docker/compose/tags

选择对应版本，amd=x64_86=x86 ，amd并不是指amd的cpu，两者没有半毛钱关系，centos通常选择amd或者x64_86或者x86。

[root@lgb /]# mv docker-compose-linux-x86_64 /usr/local/bin/docker-compose

[root@lgb bin]# chmod +x /usr/local/bin/docker-compose

[root@lgb bin]# docker-compose -version
docker-compose version 1.29.0, build 07737305

如果需要使用docker-compose部署应用，请先安装docker
