element-ui修改浏览器全局的默认滚动条样式

直接在全局css中加入以下代码，即可修改掉浏览器默认的滚动条（element-ui）

/**修改全局的滚动条*/
/**滚动条的宽度*/
::-webkit-scrollbar {
	width: 8px;
}
::-webkit-scrollbar-thumb {
	background-color: #eaecf1;
	border-radius: 3px;
}
/*表格*/
.el-table__body-wrapper::-webkit-scrollbar {
	width: 10px;
	height: 10px;
}
.el-table__body-wrapper::-webkit-scrollbar-thumb {
	background-color: #a1a3a9;
	border-radius: 3px;
}
