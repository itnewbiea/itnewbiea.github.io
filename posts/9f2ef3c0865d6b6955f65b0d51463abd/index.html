达布定理

定义

条件: 函数在开区间 I I I上可导;
结论: 导函数 f ′ ( x ) f'(x) f′(x)在开区间 I I I上具有介值定理 (即: f ′ ( I ) f'(I) f′(I)值域集合 是个区间);

#推论#
如果 f f f在 I I I区间上可导, 且 f ′ ( x ) f'(x) f′(x)在 I I I上不连续 则 f ′ ( x ) f'(x) f′(x)在 I I I上的间断点 一定是震荡间断点;
