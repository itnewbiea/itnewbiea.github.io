Docker开启远程访问

  1. 查找docker.service

    # 文件是在/lib目录下
find / -name "docker.service"

  2. 修改docker.service

    vim /lib/systemd/system/docker.service
# 在ExecStart后加入
-H tcp://0.0.0.0:2375 -H unix://var/run/docker.sock

  3. 重新载入Docker服务

    systemctl daemon-reload

  4. 重启docker服务

    systemctl restart docker.service

  5. 查看端口是否开启

    netstat -nlpt
