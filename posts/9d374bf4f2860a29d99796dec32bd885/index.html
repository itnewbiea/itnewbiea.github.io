sql 函数 if 和 ifnull

if函数：

if(a,b,c)

if判断，如果a满足条件，返回b，否则返回c

例子：

select 
  sid,
  count(if(score>=60,sid,null))
 from SC
 group by sid;

运行代码满足及格条件的返回学号sid，不满足条件的返回null

ifnull函数

IFNULL(expression_1,expression_2);

如果expression_1不为NULL，则IFNULL函数返回expression_1; 否则返回expression_2的结果。
