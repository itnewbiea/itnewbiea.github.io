n个字母的全排列

设str是含有n个不同字符的数组例如12345，perm（str,k,n）为str[0]~str[k]的所有字符全排序输出函数，n为str数组字符个数。以此类推，perm(str,k-1,n)处理的字符个数比perm(str,k,n)处理的字符个数少一个。假定perm(str,k-1,n)可求，对于第k个位置可以任取str[0]~str[k-1]内任意元素作为str[k]，再组合perm(str,k+1,n)得到perm(str,k,n)。   再递归之前先选取一个元素和位置k的元素交换，这一级递归完成之

算法

void perm(char str[],int k,int n) {
    int i,j;
    int temp;
    if(k==0) {
        for(i=0;i<n;i++) cout<<str[i];
        cout<<endl;
    } else {
        for(i=0; i<=k; i++){
            temp=str[k];
            str[k]=str[i];
            str[i]=temp;//任取str[i]与str[k]交换
            perm(str,k-1,n);
            temp=str[k];
            str[k]=str[i];
            str[i]=temp;//还原str[k]和str[i]得到原始数组，下次再任取
        }
    }
}
