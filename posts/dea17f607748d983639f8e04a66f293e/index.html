强转对象方法

java强转对象的时候我们一般会写成如下

Object target =map.get(key);
TargetBeanClass bean = null;
if(target !=null){
bean= (TargetBeanClass )target ;
}

强转对象可以写如下方法，target是null就返回null，不需要再判断强转的对象是否是null在转换会对应的对象来了

 public <T> T zhuanhuanduixiang(Object target) {
       return (T) target;
   }
   那么只需要在调用的地方直接使用该方法包装就可以了，保证代码的简介性可读性
