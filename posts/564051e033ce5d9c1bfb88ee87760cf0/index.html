vue全屏组件screenfull的使用

vue使用screenfull进入全屏

1.安装依赖

	npm install --save screenfull

2.在需要设置的页面导入

	import screenfull from "screenfull";

3.一个按钮，一个点击事件

//dom
      <el-button type="primary" class="button" @click="click">按钮</el-button>
//点击函数
    click() {
      // 判断是否支持
      if (!screenfull.enabled) {
        this.$message({
          message: "不支持全屏",  
          type: "warning"
        });
        return false;
      }
      screenfull.toggle();
    }
