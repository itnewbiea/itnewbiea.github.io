Python-Selenium-使用 pywinauto 实现 Input 上传文件

当前环境：Win10 + Python3.7 + pywinauto==0.6.8，selenium==3.14.1

示例代码

from pywinauto import Desktop
import os

app = Desktop()
dialog = app['打开']
dialog['Edit'].set_edit_text(os.getcwd() + '.\\example-01.jpg')
dialog['Button'].click()
