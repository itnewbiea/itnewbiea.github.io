vue elment-ui render中input为undefined input-number也没有对应值

this.tableData.col.forEach(item => {
        if (item.prop === 'count') {
          item.render = (h, params) => {
            let count = 9
            return (
              <div>{params.row.count}
                <el-input-number v-model={params.row.count}></el-input-number>
              </div>
            )
          }
        }
      })

问题描述：

获取不到对应的值  更改值更是不能实现  
将el-input-number 换成input 出现了 ‘undefined’

这样我们就要安装

babel-plugin-jsx-v-model （http://npm.taobao.org/package/babel-plugin-jsx-v-model）

安装后在.babelrc的插件下加上‘jsx-v-model’
