Mysql 增加rownum

Mysql 增加rownum 实现（极度推荐）

 

  1. mysql本身不像oracle那样使用 rownum 来显示行号的功能.但是可以使用设置变量的方式来实现这一功能.
  2. mysql使用 @ 定义用户变量, @@ 定义系统变量.
  3. 查询显示的具体代码

SELECT @rownum:=@rownum+1 AS rownum, frutas.*
FROM (SELECT @rownum:=0) r, frutas;
