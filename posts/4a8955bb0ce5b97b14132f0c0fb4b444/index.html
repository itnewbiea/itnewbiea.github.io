python3 判断列表是一个空列表

python3 判断空列表

@(python3)

有个判断列表是否为空的需求，试了好多方式，比如：

a = []
if a is not None:
    COMMAND
a = []
if a[0] is None:
    COMMAND

各种乱七八糟的逻辑，老是无法满足。其实很简单。

a = []
if a:
    COMMAND
  • 空列表等于 False，
  • 那么直接 if a 就是判断在列表不为空的时候，需要执行的命令
a = []

if len(a):
    COMMAND
  • 同理，len(a) = 0
  • 在 a 不为空的时候，执行 Command
