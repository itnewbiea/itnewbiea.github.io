数据库作业

为教学数据库中表插入规定的数据。

use teachingdb;
 insert
 into student
 values('11111','马明','计算机','01','女','2000/01/02',null)

将 student 表中所有学生的总学分加2，空值记录也参加计算。

use teachingdb;
 update student
 set totalcredit=totalcredit+2.0

将 student 表中马小燕的出生日期修改为’2000-01-22’。

 use teachingdb;
 update student
 set birthday='2000-01-22'
 where sname='马小燕'

:删除student表中的数据

 use teachingdb;
 delete 
 from student
 where sno='11111'
