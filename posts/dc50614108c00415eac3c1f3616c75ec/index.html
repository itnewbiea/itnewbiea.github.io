SAPUI5教程——安装Hybrid Application Tools失败

问题

如果需要使用需要将Fiori打包成Hybrid移动应用，我们需要安装Hybrid Application tools.但是我们安装过程中会遇到这个问题。

对此我们知道是缺少某个包文件，比如fs-write-stream-atomic.

所以我们会执行 sudo npm install -g fs-write-stream-atomic.
进行全局安装，但是安装成功后，再次检查问题依然存在。

方案

检查你当前nodejs环境，是那个版本的，也就是说Hybird Application tools使用的是5.4.1的环境，但是你装全局包是在另一个环境比如6.0.1，这就导致你安装的文件不存在或找不到。
对此我们需要切换node环境，可以使用nvm use 5.4.1 这个命令。
