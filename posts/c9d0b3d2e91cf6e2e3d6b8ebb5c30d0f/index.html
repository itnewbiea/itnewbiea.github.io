linux中无ll命令的解决方法

有时候在Linux中输入”ll”或者“ls -l”命令会弹出（ps: 这两个命令是等价的，前者是后者的简化版）

-bash: ll: command not found  / -bash: ls -l : command not found

这其实是因为在环境变量中ll命令被注释掉了，只需要打开配置文件

vim ~/bashrc   或者   vi ~/bashrc  再或者   nano ~/bashrc
vim /root/.bashrc 或者 vi /root/.bashrc 再或者 nano /root/.bashrc

找到这一行

# alias ll='ls $LS_OPTIONS -l'

把前面的注释"#"去掉，然后更新一下环境变量即可

. ~/.bashrc  或者   source ~/.bashrc
