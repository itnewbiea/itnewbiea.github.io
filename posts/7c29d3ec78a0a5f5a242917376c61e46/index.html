1，前言

项目基于Vue2全家桶及网易云音乐 Node Api实现，网站风格简约清新，体验比较流畅。

下载地址：https://github.com/wp993080086/music-website
预览地址：music-website

2，已有功能

  • 网易云APP扫码登录 & 网易云账号登录
  • 注册
  • 音乐播放器
  • 歌单 & 详情
  • 歌手 & 详情
  • MV & mv详情
  • 排行榜
  • 搜索
  • 评论
  • 歌词

3，使用

  • Clone该项目：git@github.com:wp993080086/music-website.git
  • 安装：npm install
  • 运行：npm run serve
  • 测试环境打包：npm run test
  • 正式环境打包：npm run build
  • 同时打包正式环境 & 测试环境：npm run all
  • 修复：npm run lint
