1、1、2、3、5、8、13、21、34求第n位数

这个是斐波那契数列，他的特点是：第0项是0，第1项是第一个1，这个数列从第三项开始，每一项都等于前两项之和。话不多说直接看代码。

1.首先在类中定义一个静态方法。

 public static void getNumber(int n) {
        int i = 1;
        if (n < 3) {
            System.out.println("1,1");
        } else if (n >= 3) {
            int a = 1, b = 1, c = 0;
            System.out.print(a + "    " + b + "    ");
            while (i <= n) {
                c = a + b;
                a = b;
                b = c;
                System.out.print(c + "    ");
                i++;
            }
        } else if (n < 0) {
            System.out.println("输入数字不符合要求");
        }
    }
