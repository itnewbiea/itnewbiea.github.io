vue element ui表格高度加滚动条自适应，纯css解决方案

解决方案：如下图

父盒子：{
 position: relative;
}
表格外层div{
 width: 100%;
    height:100%;
    display: flex;
    flex-direction: column;
    position: absolute;
}
 .el-table {
      flex: 1;
      overflow-y: auto;
    }

这样就可以了，完美自适应,另外table要加高度，分页直接放在后面即可
