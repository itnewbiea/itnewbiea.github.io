1.1 状态方程

对 LTI 离散系统：
x ( k + 1 ) = A x ( k ) + B u ( k ) x ∈ R n , u ∈ R p x(k+1) = Ax(k) + Bu(k)\quad x\in\R^n, u\in R^p x(k+1)=Ax(k)+Bu(k)x∈Rn,u∈Rp

对传统控制系统，连续系统是好处理的，离散系统是要额外考虑其他因素的
对MPC，则是反过来，离散系统是好处理的，连续系统是要额外考虑其他因素的
