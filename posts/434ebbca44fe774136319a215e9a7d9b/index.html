doLayout 解决 elementui table加载数据之后错位问题

有的时候在使用elementui 的table时候，更新完数据之后。表格就会错位。
这个时候只需要使用doLayout重新加载一下table就好了。

<el-table ref="tableref" ></el-table>
 
 //当页面数据加完完毕后
 t.$nextTick(() => {
 		// 初始化表格
          t.$refs.tableref.doLayout();
 })
