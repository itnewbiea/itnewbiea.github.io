输入一个正整数n，将n分解成质因数。例如: 输入6，则输出 6=2*3;输入 90，则输出-2*3*3*5。...

以下是一个 Python 代码示例，可以将输入的正整数n分解成质因数，并以类似于题目所示的形式输出：

def prime_factors(n):
    factors = []
    divisor = 2
    while divisor <= n:
        if n % divisor == 0:
            factors.append(divisor)
            n //= divisor
        else:
            divisor += 1
    return factors
