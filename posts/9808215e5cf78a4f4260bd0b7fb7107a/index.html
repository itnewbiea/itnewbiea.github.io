redis 数据导出及导入

本操作在linux-示例在centos6.x上操作：

redis 数据导出

  1. 命令进入redis，在ssh上输入
    redis-cli -a password #个人设置的密码
  2. 在redis上输入，先把当操作保存
    bgsave
  3. 然后停止redis
    service redis stop
  4. 进入rdb文件路径，复制dump.rdb
    cd /var/lib/redis/
  5. 重新启动redis
    service redis start

redis 数据导入

  1. 导入则是先停redis再替换此文件
    service redis stop
  2. 把复制的dump.rdb 上传到以下路径
    默认路径在 /var/lib/redis/，若非默认请自己查找
    /var/lib/redis/
  3. 执行bgsave
  4. 启动redis
    service redis start
  5. 确认redis的数据是否正常
  6. 只要有dump.rdb 文件在 可以重复执行，注意不要随便删除
