处理二手买卖、废品回收小程序 在app.json中声明permission scope.userLocation字段 教程说明

sitemapLocation
指明 sitemap.json 的位置；默认为 ‘sitemap.json’ 即在 app.json 同级目录下名字的 sitemap.json 文件

找到app.json这个文件 把这段代码加进去，保存，重新编译一下就好了

  "permission": {
    "scope.userLocation": {
      "desc": "你的位置信息将用于小程序位置接口的效果展示" 
    }
  },
