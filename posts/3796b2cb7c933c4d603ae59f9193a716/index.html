构造三栏布局

  • 三栏布局

题意：

写一个三栏布局，中间自适应，两边固定。

方法一：定位

1.使用相对定位处理两边的盒子；

2.对中间盒子的宽度不限定；

3.控制中间盒子margin使其不被挡住。

方法二：浮动

1.使用左右浮动处理两边盒子；

2.对中间盒子的宽度不限定；

3.控制中间盒子margin使其不把两边盒子挡住。

注意：浮动盒子在包含块中排列会避开常规流；常规流盒子会无视浮动盒子。

方法三：flex布局

1.使主体成为一个弹性盒子；

2.固定两边盒子宽度；

3.设中间盒子flex:1，使其它盒子等分剩余空间。

    flex:1;

    display: flex;

方法四：双飞翼布局（负边距法）

理解：1.写三个盒子，且主体盒子必须写在最上面，其中主体栏的盒子在其基础上再嵌套一个父元素盒子，且宽度撑满；

           2.使左盒子、右盒子及中间盒子的父元素盒子左浮动；

可以观察到中间盒子的父元素盒子独占一行，其余两个盒子都在下一行。

           3.使左盒子的margin-left为-100%（使其移动了父元素宽度）；

           4.使右盒子的margin-left为其自身宽度的负数；

可以观察到左盒子跑到了页面左上角，右盒子跑到了页面的的右上角。

           5.给主体栏盒子设置margin使其不被遮挡。

方法五：圣杯布局

——大体思路和双飞翼布局相同

1.三个元素均设置为浮动；

2.设置中间部分padding，给两侧留出空间；

3.再分别设置左边框与右边框，采用margin为负来使其处于同一行；

4.期间要处理浮动坍塌，及使用定位来使其合理布局。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>圣杯布局</title>
</head>
<style>
    .sanlan{
        background-color: chocolate;
        overflow: hidden;

        background-color: brown;
    }
