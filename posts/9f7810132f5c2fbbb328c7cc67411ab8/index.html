Java ++a和a++的区别

package day05；
public class Add{

	public static void main(String[] args) {
		
		  int a =0;                                 
		  int b=0;
		  int c =a++;
		  int d =++b;
		  System.out.println(c);
		  System.out.println(a);
		  System.out.println(d);
		  System.out.println(b);  
	}

运行结果如下

a++和++a 都属于自增运算符，区别是对变量a的值进行自增的时机不同。
a++是先进行取值，后进行自增。++a是先进行自增，后进行取值；
