在Android Studio 中运行React Native 项目

项目根目录执行命令安装开发依赖

yarn

检查项目SDK、NDK、JDK否配置正确

点击 Android Studio 里点击大象

全部下载完毕，点击运行按钮，编译项目

连接真机的两种方式

无线连接

adb devices
adb tcpip 5555 #连接端口默认5555
adb connect 192.168.0.102:5555	#要连接的真机的IP地址
adb -s 192.168.0.102:5555 reverse tcp:8081 tcp:8081
npx react-native start

USB连接

adb  reverse tcp:8081 tcp:8081
npx react-native start
