laravel相关开发框架

CMS（内容管理系统）：

Bootstrap CMS —— Laravel 5.1驱动的功能强大的CMS（已停止更新）

October —— 基于Laravel 5，致力于让开发工作变得简单的CMS（最高支持版本：5.5）

PyroCMS —— MVC架构的PHP内容管理系统，3.0以前基于CodeIgniter，目前基于Laravel（最高支持版本：5.8）

LavaLite —— 基于Laravel & Bootstrap 3的内容管理系统（最高支持版本：5.8）

TypiCMS —— 基于 Laravel 构建的、支持多语言的内容管理系统（最高支持版本：6.0）

Laravel and AngularJS CMS —— 基于Laravel 5.1和AngularJS的CMS（长期未更新）

Microweber —— 基于 Laravel 拖拽式生成 CMS 及在线商店利器（最高支持版本：5.4）

AsgardCMS —— 基于 Laravel 构建的、支持模块化和多语言的CMS（最高支持版本：5.5）

CoasterCMS —— 基于Laravel 构建的下一代CMS（最高支持版本：5.4，长期未更新）

BorgertCMS —— 基于 Laravel 开发的开源模块化 CMS（最高支持版本：5.6）

WebEdCMS —— 基于 Laravel 开发的开源CMS系统（最高支持版本：5.5，已停止更新）

CRM（客户关系管理系统）：

Flarepoint —— 基于 Laravel 构建的免费开源CRM平台（最高支持版本：5.4）

Monica —— 基于 Laravel 构建的开源个人关系管理应用（最高支持版本：5.8，个人 CRM）

论坛：

Laravel.io —— Laravel.io 开发者社区源码（最高支持版本：5.7）

Flarum —— 免费的、开源的、专注于简约的论坛系统，esoTalk和FluexBB的联合继承者（最高支持版本：5.7）

PHPHub —— 基于Laravel 4.2，积极向上的 PHP & Laravel 开发者社区（长期未更新）

zhihu-app —— 基于 Laravel 5.3 +Vue 构建的带前后台的、类似知乎的问答社区（长期未更新）

Hifone —— 基于 Laravel 5.2 的开源论坛系统（长期未更新）

社交网站：

ThinkSNS+ —— 基于 Laravel + Vue.js 开发的全新社交系统（最高支持版本：5.7）

Voten —— 基于 Laravel + Vue.js 构建的类似 Reddit 的实时社交书签系统（最高支持版本：5.8）

Pixelfed —— 基于 Laravel 构建开源照片分享平台（最高支持版本：5.8）

电商：

Antvel —— 基于 Laravel 5.* 开发的开源电子商务项目（最高支持版本：5.6，长期未更新）

Bagisto —— 基于 Laravel + Vue 构建的开源电商系统（最高支持版本：5.6）

Laracom —— 基于 Laravel 构建的开源在线商城系统（最高支持版本：5.7）

在线教育：

MeEdu —— 基于 Laravel 框架开发的在线视频点播系统（最高支持版本：5.7，知识付费）

BookStack —— 基于 Laravel 开发的知识管理与服务平台（最高支持版本：6.0）

项目管理：

92five app —— 基于 Laravel 框架 & Backbone JS 构建，是一个自托管的、基于web的项目管理应用（最高支持版本：4.0，长期未更新）

Scrumwala —— 基于Laravel5，项目管理应用（最高支持版本：5.1，长期未更新）

Ribbbon —— 基于Laravel 5.1和Vue.js构建的开源项目管理系统（最高支持版本：5.1，长期未更新）

Goodwork —— 基于 Laravel + Vue.js 开发的开源项目管理和团队协作项目（最高支持版本：6.0）

博客：

Canvas —— 基于Laravel 开发的轻量级博客系统（最高支持版本：6.0）

Katana —— 静态博客/站点生成器，支持 Markdown 和 GitHub Pages（非 Laravel 项目，长期未更新）

Laravel 5 Blog —— 基于 Laravel 5 开发的博客系统（最高支持版本：5.1，长期未更新）

Vuedo —— 基于 Laravel 和 Vue.js 构建的博客平台（最高支持版本：5.2，长期未更新）

Wardrobe —— 专注于写作的最小化博客平台（最高支持版本：5.7，长期未更新）

MyPersimmon —— 基于 Laravel 5.4 开发的博客系统（长期未更新）

娱乐：

Koel —— 基于 Laravel & Vue.js 开发的酷炫音乐流媒体应用（最高支持版本：5.8）

Flox —— 基于 Laravel 开发的电影、电视剧、动漫视频应用（最高支持版本：5.3，长期未更新）

后台模板：

Laravel Angular Admin —— 基于 Laravel + Angularjs + Bootstrap + AdminLTE 实现的后台模板（最高支持版本：5.2，长期未更新）

其他：

Laravel.com —— Laravel官网源码（最高支持版本：5.6）

Laravel Tricks —— Laravel小技巧&小贴士（最高支持版本：4.1，长期未更新）

Invoice Ninja —— 基于Laravel构建的开源发票及时间跟踪应用（最高支持版本：5.5）

Paperwork —— 基于Laravel 4.2 的开源的笔记&归档工具（已经处于休眠状态，新版本不再依赖 Laravel 框架）

Cachet —— 基于Laravel 5的开源的状态页系统（最高支持版本：5.7）

StyleCI —— 由 PHP CS Fixer开发，提供PHP代码风格持续集成服务（不再开源）

Podcastwala —— 基于Laravel 5，构建属于你自己的播客网站（最高支持版本：5.0，长期未更新）

Deployer —— 基于Laravel 开发的免费开源 PHP 应用部署工具（最高支持版本：5.5）

RSS Monster —— 基于 Lumen 开发的 RSS 聚合器和阅读器（已改用 Vue + Express 实现）

Attendize —— 基于 Laravel 框架开发的开源门票及活动管理应用（最高支持版本：5.6）

Notadd —— 基于 Laravel 的下一代 PHP 开发框架（API+SPA 单页应用 ）（已调整为 Next.js 微服务框架）

Blockscloud —— 基于 Laravel + Vue 实现的 Web OS 系统（长期未更新）

Akaunting —— 基于 Laravel 构建的免费在线财务会计系统（最高支持版本：5.4）

Unifiedtransform —— 基于 Laravel 开发的开源校园管理系统（最高支持版本：5.5）

Pterodactyl Panel —— 基于 Laravel + Node.js + Go 构建的开源游戏服务器管理系统（最高支持版本：6.0）

WebStack —— 基于 Laravel 开发的开源网址导航项目（最高支持版本：5.5）

Crater —— 基于 Laravel 开发的免费开源发票应用（最高支持版本：6.0）

Cipi —— 基于 Laravel 开发的云服务控制面板（最高支持版本：7.x），支持 AWS、DigitalOcean、 Google Cloud、Linode 等
