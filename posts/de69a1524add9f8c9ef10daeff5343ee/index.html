前言

本文是以kafka_2.6.0为模板

  • 所有的topic命名test

1. 创建topic(–create)

bash kafka-topics.sh --create --zookeeper localhost:2181 --replication-factor 2 --partitions 3 --topic test
  • zookeeper： 定义 zookeeper 集群服务器地址，如果有多个 IP 地址使用逗号分割，一般使用一个 IP 即可
  • replication-factor： 定义分区副本数，1 代表单副本，建议为 2 partitions：定义分区数
  • topic： 定义 topic 名称

2. 查看当前服务器中的所有 topic(–list)

kafka-topics.sh --list --zookeeper localhost:2181

3.查看某个 topic 的详情(–describe)

bash kafka-topics.sh  --describe --zookeeper localhost:2181

4.发布消息(–broker-list )

bash kafka-console-producer.sh --broker-list localhost:9092 --topic test

5.消费消息(–bootstrap-server)

bash kafka-console-consumer.sh --bootstrap-server localhost:9092 --topic test --from-beginning

6.修改分区数

bash kafka-topics.sh --zookeeper localhost:2181 --alter --topic test --partitions 6

7.删除 topic

bash kafka-topics.sh --delete --zookeeper localhost:2181--topic test
