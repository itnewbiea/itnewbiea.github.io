GSEP 1级样题 闰年统计

题目描述

小明刚刚学习了如何判断平年和闰年，他想知道两个年份之间（包含起始年份和终止年份）有几个闰年。你能帮帮他吗？

输入格式

输入一行，包含两个整数，分别表示起始年份和终止年份。约定年份在 1到 2022 之间。

输出格式

输出一行，包含一个整数，表示闰年的数量。

样例 #1

样例输入 #1

2018 2022

样例输出 #1

1

样例 #2

样例输入 #2

2000 2004

样例输出 #2

2

解题思路

闰年是指能被4整除但不能被100整除的年份，或者能被400整除的年份。
遍历从起始年份到终止年份的每一年，对每一年的情况进行判断，如果是闰年就计数器加1。

参考代码1：

#include <iostream>
using namespace std;
int main()
{
	int m,n,i,cnt=0;
	cin>>m>>n;
	for(i=m;i<=n;i++){
		if(((i%4==0)&&(i%100!=0)||(i%400==0))){
			cnt++;
		}
	}
	cout<<cnt;
 	return 0;
}
