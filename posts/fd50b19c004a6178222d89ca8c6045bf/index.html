项目初始化报 404 Not Found - GET https://registry.npmjs.org

npm install报 404 Not Found - GET https://registry.npmjs.org

解决办法：
1.先查看当前的代理设置

npm config get proxy

2.如果代理设置时null,则需要配置新的代理地址

npm config set registry 新的地址
根据情况 是公司项目的地址还是默认的国内cnpm淘宝镜像（https://registry.cnpmjs.org/）
