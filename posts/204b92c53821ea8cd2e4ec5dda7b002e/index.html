解决JsonResponse接口返回值中文是乱码的情况

1.初学Django，写了一个接口，但是返回值中文为乱码

2.views.py代码为

from django.http import JsonResponse
from django.views.decorators.csrf import csrf_exempt



@csrf_exempt
def test_api(request):
    return JsonResponse({"result": 0, "msg": "成功"})

3.解决方法，

JsonResponse({"result": 0, "msg": "成功"},charset='utf-8')
from django.http import JsonResponse
from django.views.decorators.csrf import csrf_exempt

@csrf_exempt
def test_api(request):
    return JsonResponse({"result": 0, "msg": "成功"},charset='utf-8')
