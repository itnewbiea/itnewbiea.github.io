使用 el Descriptions 描述列表时前端报错 - did you register the component correctly

报错原因：

element-ui版本问题。是到了element-ui 2.15.6才开始更新上线的，而项目里的element-ui是 2.15.6以下的版本，所以使用报错。

解决方案：

将项目内element-ui更新到最新版本

1.卸载现有element-ui版本

npm uninstall element-ui

2.安装最新版element-ui

npm i element-ui -S

3.重新安装依赖

npm install
