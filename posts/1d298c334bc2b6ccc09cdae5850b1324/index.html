关于vue项目行内式添加背景图片的可行方案

在Vue中，你可以使用v-bind指令来动态绑定行内样式。要添加背景图片，你可以使用background-image属性，并将其值设置为图片的URL。如果你想同时添加多个class，可以使用数组语法。

<template>
  <div :style="{'background-image': 'url(' + imageUrl + ')'}" :class="[class1, class2]">
    <!-- 内容 -->
  </div>
</template>

<script>
export default {
  data() {
    return {
      imageUrl: require("../../assets/images/"),
      class1: 'class1的名称',
      class2: 'class2的名称'
    };
  }
};
</script>

值得注意的是在data中的url引入要写commn.js的引入方式去实现
