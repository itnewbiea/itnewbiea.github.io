python--如何进行去重

文章目录

  • 一、自定义函数去重
  • 二、内置函数去重

一、自定义函数去重

解析思路：
1、确定去重目标
2、给一个空列表去接收去重后的元素
3、遍历需要去重的序列，将重复数据过滤
4、打印去重后的数据

l = [1,1,3,2,2,3,4,2,5]
new = []
for i in l:
    if i not in new:
        new.append(i)
print(new)

输出结果：

[1, 3, 2, 4, 5]

二、内置函数去重

l = [1,1,3,2,2,3,4,2,5]
b = list(set(l))
print(b)

输出结果：

[1, 2, 3, 4, 5]

可以看出去重后改变了序列的顺序，所以去重后需要按元素索引进行排序，才会保持序列的原有顺序

代码如下：

l = [1,1,3,2,2,3,4,2,5]
a = list(set(l))
a.sort(key=l.index)
print(a)

输出结果：

[1, 3, 2, 4, 5]
