Vim设置支持鼠标，光标定位，滚轮上下移动

本章主要说明如何使用鼠标功能。


vim打开鼠标（临时设置）

:set mouse=a


长久设置，在~/.vimrc中添加


set mouse=a


关闭鼠标 

:set mouse-=a


mouse参数说明

  1. // n  普通模式  
  2. // v  可视模式  
  3. // i 插入模式  
  4. // c 命令行模式  
  5. // h 在帮助文件里，以上所有模式  
  6. // a 以上所有模式  
  7. // r 跳过|lit-enter|提示  
  8. // A 在可是模式下自动选择
