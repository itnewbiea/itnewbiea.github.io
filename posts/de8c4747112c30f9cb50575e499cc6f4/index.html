C语言：求1-3+5-7 +--99 + 101的值。（两种方法)

方法1: (1 +5+9+13+17 +.... +101)-(3+7 +11 +15 +....+99)程序源代码:

#include<stdio.h>
void main()
{
    int i,j,sum1=0,sum2=0,sum;
    for(i=1;i<=101;i+=4)
        sum1 += i;
    for(j=3;j<=99;j+=4)
        sum2 +=j;
        sum=sum1-sum2;
        printf("1-3+5-7+....-99+101=%d\n",sum);
}
方法2:符号交替变换程序源代码:

#include<stdio.h>
void main()
{
    int i,sum=0,s=1;
    for(i=1;i<=101;i+=2)
    {
        sum =sum+i*s; /*控制符号交替变换*/
        s=-s;
    }
        printf("1-3+5-7+....-99+101=%d\n",sum);
}
运行结果：

1-3+5-7+....-99+101=51
