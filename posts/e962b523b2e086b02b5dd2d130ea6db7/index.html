JS中把字符转成ASCII值的函数

字符转ascii码：用charCodeAt();
ascii码转字符：用fromCharCode();

看一个小例子

<script>
str="A";
code = str.charCodeAt();
str2 = String.fromCharCode(code);
str3 = String.fromCharCode(0x60+26);

document.write(code+'<br />');
document.write(str2+'<br />');
document.write(str3);
</script>

输出：

65
A
z
