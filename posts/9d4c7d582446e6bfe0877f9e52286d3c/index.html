HTML复习笔记汇总(1)

这篇文章主要写一下，复习HTML时候觉得有必要要记下来的东西

一，HTML文档的基本结构

1,一个简单的html页面例子

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">

    <html>

    <head>

    <meta http-equiv="Content-Type" content="text/html;charset=utf-8">

    <title>html标签</title>

    </head>

    <body>

    一个html页面

    </body>

    </html>

2,html中的标准属性

(1):calss:表示特定元素所属的一个类或组

(2):id:为元素指定一个唯一性的标志符

(3):style:为元素指定CSS属性，这称为内联样式

(4):title:为元素提供一个文本标题，

语言属性: dir 把文本的阅读方向设置为值ltr(从左到右)或rtl(从右到左)所指定的方向  lang 指定用于书写所包含的内容的语言

3,html字符实体

  显示结果:实体名称    ：&nbsp   <:&lt   >:&gt    &:&amp  ":&quot   ':&apos  分:&cent  磅:&pount  元:&yan  节:&sect  版权:&copy  注册商标:&reg  乘号:&times  除号:&divide

4,为文档添加注释

<!--    注释内容   -->

注意注释只是在浏览器中不显示，但是在客户端的源代码中仍是可以看到的

二:辅助性元素

1，用<title>标签为文档添加标题

例子:<title>这是一个文档标题</title>

2，用<base>标签为文档设置基础URL

例子:<base href="http://www.myweb.com/" />

3，用<link>标签定义文档关系链接

例子:<link rel="stylesheet" type="text/css" href="main.css" />

4，用<style>标签为文档创建内部样式表

5，用<script>标签添加脚本程序

例子:<script type="text/javascript">

window.onload = function(){

alert("hello world")

};

例子:<script type="text/javascript" src="main.js"></script>

6，用<meta>标签为文档定义元数据

例子:<meta name="name属性可用的关键字" content="关键字可用的值">

        <meta http-equiv="http-equiv属性可用的关键字" content="关键字可用的值">

name属性值

  • keywords  content="HTML CSS javascript",    用来描述该文档的关键字，为搜索引擎提供线索，非常有用
  • Description content = "这是我的第一个网页，你来收录吧"  为搜索引擎提供一份关于文档的简单说明，非常有用
  • Author content="xiaobaicai"  用来说明文档的设计者，很少用
  • Generator content="Dreamweaver"  用来说明创建网页所使用的编程工具，很少用
  • Data content="2018-5-8" 用来说明文档的创建日期，不常用
  • Copyright content="某某出版社"  用来说明文档的版权所有，不常用
  • Robots Content="all none index noindex follow nofollow"  用来定义搜索引擎的搜索方式

        搜索方式:

  •  index：允许搜索到的该网页收录到搜索引擎数据库，noindex反之
  • follow : 即允许搜索引擎机器人通过该网页的链接链接到其他网站，nofollow反之
  • all: 如果上述都允许，写出all none反之

http-equiv属性的值

  • Content-type  content="text/html;charset=gb2312"  设置网页的内容类型和使用的字符集
  • Content-language content="zh-CN",   描述本网页使用的语言，浏览器根据此项可用选择正确的语言渲染特点
  • Refresh content="60,url=new.html"  设置浏览器定时跳转，或者一定时间后刷新自身
  • Pragma content="no-cache" 设置浏览器从本地缓存中调阅页面内容，设定后一旦离开网页就无法从本地缓存中再调出
  • Cache-Control content="no-cache" 作用和pragma的作用一样，只不过针对的http版本不同
  • Expires content="Tue,24,jan,2013,16:26:32 GMT" 设置网页的到期时间，一旦到期则必须到服务器中从新调用
  • set-cookie content="Tue,24,Jan 2013 16:26:32 GMT" 设置cookie的过期时间
  • windows-Target content="_top" 可用设置将当前网页加载到哪个框架中
