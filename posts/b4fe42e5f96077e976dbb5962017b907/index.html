VsCode常用快捷键

  1. 全屏：F11
  2. zoomIn/zoomOut：Ctrl +/-
  3. 侧边栏显/隐：Ctrl+B
  4. 控制台终端显示与隐藏：ctrl + ~
  5. 显示资源管理器 ：Ctrl+Shift+E
  6. 自动换行 : alt + z
  7. 快速回到顶部 ： ctrl + home
  8. 快速回到底部 : ctrl + end
  9. 显示搜索： Ctrl+Shift+F
  10. 显示 Git： Ctrl+Shift+G
  11. 显示 Debug ：Ctrl+Shift+D
  12. 显示 Output ：Ctrl+Shift+U
  13. 全局查找 ：Ctrl+Shift+F
  14. 万能键：F1
  15. 找到所有的引用： Shift+F12
  16. 同时修改本文件中所有匹配的： Ctrl+F12
  17. 重命名：比如要修改一个方法名，可以选中后按 F2，输入新的名字
  18. 回车，会发现所有的文件都修改了
  19. 跳转到下一个 Error 或 Warning：当有多个错误时可以按 F8 逐个跳转
  20. Shift + Alt + F 格式化代码
  21. F12 跳转到定义处
  22. Ctrl + ` 打开集成终端
  23. Ctrl + Shift + ` 创建一个新的终端
