StatefulWidget initState()方法

这个方法的使用一定要注意一点 不然在渲染页面的时候 不会第一个走initState方法

   @override
  void initState() {
    // TODO: implement initState
    super.initState();
    _getList("服务商家"); //先初使用化数据
  }

####要注意的是 一定要调用super.initState()方法 不然的话 界面渲染的时候不会先走这个方法，不调用super的话 会在build之后走这个方法
