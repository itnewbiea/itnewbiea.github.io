docker 配置本地仓库才能拉取仓库镜像&目录挂载

一、编辑daemon

vim /etc/docker/daemon.json

{
"insecure-registries": ["192.168.160.70:5000"]
}

重启docker服务

systemctl restart docker.service

docker目录挂载

-v=源文件地址：目的地址
-v/–volume 含义一致

rw 读写权限 ro 只读 默认读写权限

demo:

docker run --name=myName --volume="源文件:镜像中文件地址:ro"  hostname="myHostName"
镜像名称

通过 dokcer inspect CONTAINER ID 命令进行查看挂载情况
