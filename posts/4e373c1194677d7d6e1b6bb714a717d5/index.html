在移动硬盘上装Ubuntu

需要工具

  1. U盘（作为系统启动器）
  2. 大存储移动硬盘
  3. Ubuntu.iso

步骤

  1. 首先格式化移动硬盘
    1.1 cmd ：diskpark
    1.2 diskpart： select disk X (选择你的移动硬盘)
    1.3 diskpart： clean
    1.4 在计算机磁盘管理新建简单卷
  2. 装系统
    2.1 使用USB writer 将.iso文件写入U盘
    2.2 开机按F10（不同品牌的机器不同）进入BLOS，选择U盘启动
    2.3 安装类型，选择移动硬盘，取文件类型Ext4，根结点“\”，设置你的密码 ，语言最好设置为英文。中文可能有很多意料之外的错
  3. 装机准备
    Terminal ：
    sudo apt-get update
    sudo apt-get upgrade
    sudo ubuntu-drivers autoinstall
    (系统自行装显卡驱动，不要自己装，真的很麻烦，很容易出错，我一直用的系统自己装的，一直没问题)
