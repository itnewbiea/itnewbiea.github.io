HTML基本语法学习

一、HTML简介

HTML的英文全称是 Hyper Text Markup Language，即超文本标记语言
最新的版本是HTML5，H5中新增了对音视频的支持。HTML是一个弱语言(对语法的要求不严格)。

www简称w3c，称为万维网联盟，该机构是制定web技术规范

.

二、HTML基本结构

1、网页基本结构
<!DOCTYPE>:
    <!DOCTYPE> 位于文档的第一行，用于向浏览器说明当前文档使用的是哪种 HTML 规范。
    <!DOCTYPE html> 表示的是本文档 使用/遵循 HTML5 规范。

<html>:是紧接 <DOCTYPE> 之后的元素，它也被称为 根元素，因为它是整个 HTML 文件层次结构的顶点。
     该元素以 <html> 标签开始，以 </html> 标签结束，其中『包含』文档的头部和体部。
         
<head>:用于定义 HTML 文档的头部信息，紧跟在 <html> 元素之后，主要用于封装其它位于文档头部的标记
     <meta charset="UTF-8">:设置浏览器以哪种编码解析网页，编码不对网页内容会乱码
     <meta name="viewport" content="width=device-width, initial-scale=1.0">:可视区域设置
    <meta name="keywords" content="Java学习" />：设置搜索引擎关键字
    <title>网页标题</title>:设置网页的标题
        
<body>:用于定义 HTML 文档所要显示的内容。
<body> 元素中的信息才是最终展示给用户的信息。一个 HTML 文档中只能有一个<body> 元素。

7、列表标签

列表标签有三种：有序列表，无序列表（常用），描述列表（常用）

  • 有序列表
  语法：
  <ol type="A">
      <li>新闻</li>
      <li>游戏</li>
      <li>电影</li>
      <li>音乐</li>
  </ol>

  属性type:
     更改列表前的序号（普通数字，大写/小写字母，罗马数字）
  • 无序列表
语法：
<ul type="disc">
    <li>游戏</li>
    <li>电影</li>
    <li>音乐</li>
</ul>

属性type:
   更改列表前的序号
   空心圆:circle
   实心圆:disc
   黑色方块:square
  • 描述列表
    描述列表一般用于图文混合
语法：
<dl>
	<dt>放图片或标题</dt>  //一般一个dl中只会出现一个dt
	<dd>描述的内容</dd>  //一般dl中可以出现多个dd
</dl>
8、图片标签

图片标签是用来在网页中显示图片的，一般将图片保存到当前项目的images目录中

语法：
 <img  src="图片的地址" alt="当图片无法显示时的提示文字" title="鼠标放到图片上时的提示文字"  width="图片的宽度"  height="图片的高度"/>

​ 资源的路径分为两种：绝对路径，相对路径
​ 绝对路径：资源在硬盘上的完整路径（d:/image/01.jpg）
​ 相对路径：你要访问的资源相对于当前网页的路径(…/:返回上一级 ./:表示当前目录，一般不写)

10、表格标签

1、基本表格

在现代网页中，表格标签只用来进行数据的显示
在现代网页中，表格标签只用来进行数据的显示

语法:
<table border="1" cellspacing="0" cellpadding="0" align="center">
	<tr>
	   <td></td>
	   <td></td>
	   ...
	</tr>
	...
</table>

标签说明：

​     table:表示要绘制一个表格

​     tr:表格中的一行

​      td:行中的一个单元格

​     th:行中的一个单元格，一般用在第一行的表头上，自带一些样式(字号更大，加粗)



属性说明：

​     border:是否显示表格的边框，0：不显示  1：显示

​    cellspacing：设置单元格与单元格之间的距离，数字越大，距离越大

​     cellpadding:单元格的边与单元格中内容之间的距离

​    align:对齐方式，写在table标签上时，会将更改整个表格在页面上的位置(靠左，靠右，居中)。

 	   写在tr上时，会修改整行数据的位置

​          写在td上时，只会修改当前单元格中数据的位置

2、表格结构化标签

caption：表示表格的标题，会在表格的正中显示
thead：表示表格的头部,表格的表头部分
tbody：表示表格的主体。
tfoot：表示表格的底部

示例：

<table border="1" cellspacing="0" cellpadding="5" align="center">
        <!-- 设置表格标题 -->
        <caption>
            <tr>
                <td>
                    用户信息
                </td>
                <td></td>
                <td></td>
                <td></td>
            </tr>
        </caption>
        <!-- 表格头部 -->
        <thead>
            <tr>
                <th>编号</th>
                <th>姓名</th>
                <th>年龄</th>
                <th>性别</th>
            </tr>
        </thead>
        <!-- 表格的数据部分 重要-->
        <tbody>
        <tr align="center">
            <td>1</td>
            <td>小白</td>
            <td>13</td>
            <td>男</td>
        </tr>
    </tbody>
</table>

4、表单标签【重要】

表单标签是网页与用户进行交互的一组标签

语法：所有的表单标签都必须写在<form>标签中
 <form action="要将数据提交给哪个服务器" method="get/post">  //method:表单的提交方式
     各种表单标签
 </form>

  • 普通文本框

    普通文本框中可以输入任何内容

    <input type="text" name="名字" placeholder="文本框中没有内容时的提示文字">  //name中的名字不能重复

属性说明：
   name:表单的名称，该名称是给java程序获得这个表单内容的key
   placeholder:文本框中没有内容时的提示文字
   value:为表单设置默认值，如果输入新值，则覆盖

  • 密码框

    用来输入密码

    <input type="password" name="名字" placeholder="文本框中没有内容时的提示文字">  
//name中的名字不能重复

所有属性同文本框

  • 单选按钮

    多个单选按钮只能选中一个，前提是取相同的name名称

    <input type="radio" name="sex" value="1" checked="checked">男
<input type="radio" name="sex" value="0">女

    属性说明：

    name:提交数据的key,表示多个单选按钮是一组，保证只能选择一个

    value:默认值，该值一般和数据库中对应的值一样(提交到数据库中的值)

    checked:设置单选按钮默认被选中

  • 复选框（多选按钮）

    <input type="checkbox" name="classname" value="JAVA" checked="checked">java
<input type="checkbox" name="classname" value="MYSQL">mysql
<input type="checkbox" name="classname" value="PHP">php
<input type="checkbox" name="classname" value="HTML">html

  • 文件域

    用来选中要上传的文件

    <input type="file" name="名称">

  • 文本域【了解】

    文本域可以一次输入多行内容

     <textarea rows="10" cols="10">显示的内容写在这</textarea>
 属性说明：
    rows:高度(一共可以写多少行字)
    cols:宽度，一行可以写多少个字

  • 下拉框

    <select name="ai">
   <option value="-1">---请选择---</option>
   <option value="抽烟">抽烟</option>
   <option value="把妹">把妹</option>

</select>

    属性说明：

    option:子项，可以选择的项，可以有多个

    value:对应项的值，与数据库一致

  • 按钮

    • 提交按钮【常用】

      一个form中只能有一个提交按钮，提交按钮的作用是将form标签下所有表单的值发送给指定服务器

      <input type="submit" value="按钮上显示的文字">

    • 图片按钮

      图片按钮的作用与提交按钮相同

    • 普通按钮【常用】

      普通按钮没有任何特殊功能，需要自己写代码实现功能

10、 Marquee滚动标签
Marquee滚动标签也称为跑马灯标签。可以让图片或文字在网页上滚动显示
语法:
  <marquee scrolldelay=”ms” direction=”up|down|left|right”> 要滚动的内容 </marquee>
属性值描述用法举例
scrolldelay表示滚动延迟时间，默认值为90,可以替换为任意整数< marquee scrolldelay=“100”>我是内容< /marquee>
direction表示滚动的方向，默认为从右向左,可选值为up|down|left|right< marquee direction=“left”>我是内容< /marquee>
behavior表示移动方式 <scroll(循环移动)、slide(只移动一个回合)、alternate(来回移动)>< marquee behavior=“scroll”>我是内容< /marquee>
scrollamount表示移动速度,取值正整数,数值越大移动越快< marquee scrollamount=“50”>我是内容< /marquee>
11、多媒体标签
 多媒体标签是在HTML5.0中新添加的内容，可以直接在网页上播放音频和视频
  • 音频标签
音频用来播放音乐文件
语法:
   <audio src="play/1.mp3" autoplay=“autoplay” controls=“controls” preload=“preload”>
       你的浏览器不支持音频播放
    </audio>

说明：audio是在html5.0中添加的内容，在html5.0以下不被支持，如果浏览器不支持html5则会显示中间的提示文字

  • 视频标签
视频标签用来播放视频文件
语法:
  <video src="play/3.mp4" controls="controls">
        您的浏览器不支持 video 标签。
     </video>
