matlab中的~用法和~=

用法1：表示非（与java中的！类似）

if ~flag
count++;
end

用法2：忽略函数返回的某个参数。比如size函数有两个返回值，而我们只需要第一个参数。则

[m,n]=size(A);%返回矩阵A的行和列
[m,~]=size(A);

ps：看程序看到一段sum([1 2 3 4]~=[2 3 1 4])。分开来看 里面的~=返回的是个逻辑数组[1 1 1 0],每个值对应的是向量中元素是否相等，相等为0，否则为1.

整个意思就是得到两向量中不同元素的个数。在命令行输入验证

>> [1 2 3 4]~=[2 3 1 4]

ans =

  1×4 logical 数组

   1   1   1   0

>> sum([1 2 3 4]~=[2 3 1 4])

ans =

     3
