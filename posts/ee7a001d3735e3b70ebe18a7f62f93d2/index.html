1. NVIDIA驱动失效简单解决方案：NVIDIA-SMI has failed because it couldn‘t communicate with the NVIDIA driver.

参考链接

  1. 使用nvidia-smi监视工具查看图形卡的状态出现如题的错误

    nvidia-smi

  2. 检查驱动和cuda

    nvcc -V

  3. 查看已安装驱动的版本信息

    ls /usr/src | grep nvidia

    比如我的驱动版本是：nvidia-450.57

  4. 依次输入以下命令

    sudo apt-get install dkms

    sudo dkms install -m nvidia -v 450.57

等待安装完成后，再次输入nvidia-smi，查看GPU使用状态

2.在Ubuntu 20.04上安装NVIDIA驱动程序的方法

参考链接

  1. 获取可用驱动信息

    ubuntu-drivers devices

    前面找driver，后面找recommend发现了系统推荐安装的驱动程序nvidia-driver-470

  2. 安装驱动

    sudo apt install nvidia-driver-470

  3. 安装完成后重启系统

    sudo reboot

  4. 重启成功后，使用nvidia-smi监视工具查看图形卡的状态

    nvidia-smi

3.安装nvidia驱动Perform MOK management 界面键盘失灵现象

参考链接

背景：ubuntu20.04安装nvidia驱动在Perform MOK management 界面，键盘失灵，导致安装驱动失败。
原因：没有关闭 secure boot；

  1. 开机狂按F2或者进入systerm setup，将secure boot 转换为disable ;

  2. 将显卡模式转换为: dynamic graphics ；

  3. 之后安装nvidia 驱动，重启，进入Perform MOK management 界面，键盘就可以使用了；
