hive 查看版本号

在使用hive的时候，有时候需要查看当前的hive的版本。命令为：hive> hive.hwi.war.file

例如：

hive> set hive.hwi.war.file;
set hive.hwi.war.file
hive.hwi.war.file=lib/hive-hwi-0.12.0.war
