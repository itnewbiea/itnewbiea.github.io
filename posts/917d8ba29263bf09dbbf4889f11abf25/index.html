使用navicat访问服务器mysql时的权限配置


在MySQL的user表中，host列表示允许访问MySQL服务器的主机地址或主机名。该列指定了允许连接到MySQL服务器的来源。例如，localhost表示只允许本地主机连接，而%表示允许来自任何主机的连接。所以修改user表中的host列为%，即可远程访问服务器mysql。
1.在服务器上运行mysql 

mysql -u root -p

2.进入其中的mysql数据库

use mysql;

3.查看数据库的user和host表中的权限信息

select user from user;


select host from user;

4.修改”localhost“为“%”
我们要修改的是user列为root所对应的host列中的localhost，可输入以下代码进行修改。

update user set host = '%' where user ='root';

5.刷新mysql

flush privileges;

6.navicat连接服务器mysql

现在便可以通过navicat成功连接服务器mysql了。
