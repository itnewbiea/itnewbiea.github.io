C语言中库函数自带的排序函数qsort

之前一直不知道C语言还有默认的排序函数,而且还是快速排序的函数,当时觉得自己C语言白学了,了解之后我觉得应该把这个函数的用法和用例展示出来,供大家参考。

#include <stdlib.h>

void qsort( void *buf, size_t num, size_t size, int (*compare)(const void *, const void *) );

功能： 对buf 指向的数据(包含num 项,每项的大小为size)进行快速排序。如果函数compare 的第一个参数小于第二个参数，返回负值；如果等于返回零值；如果大于返回正值。函数对buf 指向的数据按非降序排序。

要用这个函数,我们只需要自己实现compare这个函数即可,但是要满足函数规定的功能。

下面是我的实现代码：

#include <stdio.h>
#include <stdlib.h>
#include <assert.h>

int compare(const void *value1, const void *value2);
void printArray(int *array, int size);

int main(int argc, char const *argv[])
{
    int size = 0;
    scanf("%d", &size);
    assert(size > 0);

    int *array = (int *)calloc(size, sizeof(int));
    int i = 0;
    for (i = 0; i < size; ++i) {
        scanf("%d", &array[i]);
    }

    qsort(array, size, sizeof(int), compare);
    printArray(array, size);

    free(array);
    return 0;
}

int compare(const void *value1, const void *value2) 
{
    return *(int*)value1 - *(int*)value2;
}

void printArray(int *array, int size)
{
    assert(array != NULL && size > 0);
    
    int i = 0;
    for (i = 0; i < size; ++i) {
        printf("%d ", array[i]);
    }
    printf("\n");
}

当然，这个qsort函数能实现的排序不只是对int型的数组，它还可以对浮点数、数据结构进行排序，只要你自己实现相应的compare函数即可。

默认情况下qsort函数是按非降序进行排序,但是,如果你想要让数组按非升序的方式排序,那只需把compare函数的返回值更改成与原先相反即可,也就是如果函数compare 的第一个参数小于第二个参数，返回正值；如果等于返回零值；如果大于返回负值。各位有兴趣可以尝试一下。
