C语言--从键盘中输入10个整数到数组中，统计数组中正数的个数及正数的平均值（平均值保留两位小数）

源代码：

#include <stdio.h>

int main(){
	int arr[10];
	int i=0,count=0,sum=0;
	double avg=0;
	printf("请输入10个整数：");
	for(i=0;i<10;i++){
		scanf("%d",&arr[i]);
	}
	for(i=0;i<10;i++){
		if(arr[i]>0){
			count++;
			sum=sum+arr[i];
		}
	}
	avg=(double)sum/count;
	printf("正数的个数为%d，平均值为%.2lf\n",count,avg);
	return 0;
}
