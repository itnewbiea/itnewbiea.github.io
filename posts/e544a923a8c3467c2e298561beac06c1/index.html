GitHub提交项目到仓库fatal: No configured push destination.

原因：没指定提交到哪个远程仓库

解决方法：

在你git add .和commit之后

git add .
git commit -m "信息"

git push前输入以命令

git branch -M main
git remote add origin git@github.com:xxx(你GitHub的名称)/xxx.git(你仓库的名称)
git push -u origin main
