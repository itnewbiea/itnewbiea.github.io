SecureCrt传文件的三种方式

总结下securecrt传文件的三种方式.

1. scp 
scp [-1246BCpqrv] [-c cipher] [-F ssh_config] [-i identity_file] [-l limit] [-o ssh_option] [-P port] [-S program] [[user@]host1:]file1 [...] [[user@]host2:]file2

2. ZMODEM file send/recv
rz 会出个对话框让你选择文件
sz filename

3. sftp
securecrt 按下ALT+P就开启新的会话 进行ftp操作。
