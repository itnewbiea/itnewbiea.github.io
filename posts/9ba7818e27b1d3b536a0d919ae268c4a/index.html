vs报错：线程间操作无效: 从不是创建控件的线程访问它

winform中如果是从其它线程访问控件，而不是创建控件的线程那么就会报错（编译器的跨线程访问）

方法1：禁止编译器跨线程检查

Control.CheckForIllegalCrossThreadCalls = false;

方法2：使用Action和Invoke

    FrmWelcom frmWelcom = new FrmWelcom();
    Thread thread = new Thread(() =>
    {
        Thread.Sleep(2000);
        //frmWelcom.Close();

        //【方法2】使用invoke，这个是异步invoke的意思，Action可以放到外面写
        frmWelcom.BeginInvoke(new Action(() =>
        {
            frmWelcom.Close();
        }));

    });
