解决layui弹出层关闭后，在列表下面还出现弹出层的内容

解决方法：

//页面层
layer.open({
    type: 1,
    skin: 'layui-layer-rim', //加上边框
    area: ['98%', '98%'], //宽高
    content: $("#test_oe"),
    end:function (res) {
       $("#test_oe").css("display",'none');
   }
});

end - 层销毁后触发的回调

类型：Function，默认：null

无论是确认还是取消，只要层被销毁了，end都会执行，不携带任何参数。
