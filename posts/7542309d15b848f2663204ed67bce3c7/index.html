docker部署nacos2.2.0+数据持久化配置+效果实测图

统一说明:linux服务器可以借助很多工具实行可视化编辑操作，本文使用finallshell

1.拉取nacos镜像

docker pull nacos/nacos-server:v2.2.0

2.启动镜像

docker run --name nacos \
-p 8848:8848 \
-p 7848:7848 \
-p 9848:9848 \
-p 9849:9849 \
--privileged \
--restart=always \
-e JVM_XMS=256m \
-e JVM_XMX=256m \
-e MODE=standalone \
-e PREFER_HOST_MODE=hostname \
-v nacos_logs:/home/nacos/logs \
-v nacos_conf:/home/nacos/conf \
-d nacos/nacos-server:v2.2.0


--name  nacos容器名称
-p 8848:8848代表将当前实例的nacos容器中的8848端口映射到linux的8848端口
7848、9848和9849是nacos 2.2之后需要进行开放
--privileged    容器拥有真正的root权限
--restart=always   容器启动和nacos一起启动
-e JVM_XMS=256m     JVM参数
-e JVM_XMX=256m   	JVM参数
-e MODE=standalone 	 单例运行
-e PREFER_HOST_MODE=hostname    是否支持hostname 采用默认即可
-v nacos_logs:/home/nacos/logs    将容器/home/nacos/logs目录文件夹映射到宿主机的/mydata/nacos/logs目录下
-v nacos_conf:/home/nacos/conf    外部挂载application.properties配置文件
-d nacos/nacos-server:v2.2.0   -d 代表以后台方式运行  nacos-server:v2.2.0代表用的是哪个镜像
