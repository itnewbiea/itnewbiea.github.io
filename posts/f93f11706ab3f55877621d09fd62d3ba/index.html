MegaCli64检查磁盘矩阵坏道情况

 /opt/zabbix/lldscripts/MegaRAID/MegaCli/MegaCli64 -PDList -aALL|grep -E "Slot|Error"

Media Error Count: 0 Other Error Count: 0 这两个数值来确定阵列中磁盘是否有问题；

Medai Error Count 表示磁盘可能错误，可能是磁盘有坏道，这个值不为0值得注意，数值越大，危险系数越高，

Other Error Count 表示磁盘可能存在松动，可能需要重新再插入。
