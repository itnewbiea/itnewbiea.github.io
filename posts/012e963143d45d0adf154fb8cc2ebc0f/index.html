使用方法

使用：
1、clone或下载git clone https://github.com/Dontla/pdf2word.git
2、pip install -r requirements.txt安装依赖（安装过程可能会出小问题，比如我这里说跟tensorflow-gpu冲突）
3、修改config.cfg文件并创建文件夹

4、将需要转换的pdf文档放入pdf文件夹

5、执行程序

6、查看结果：空无一物
