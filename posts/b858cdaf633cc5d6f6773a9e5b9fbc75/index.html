手动下载依赖使用 pip 安装 wxPython

1. 下载 wxPython 的依赖

下载地址：https://download.csdn.net/download/Wh1teMaster/12606562

numpy-1.18.1-cp37-cp37m-win_amd64.whl 
Pillow-7.0.0-cp37-cp37m-win_amd64.whl 
six-1.14.0-py2.py3-none-any.whl    
wxPython-4.0.6-cp37-cp37m-win_amd64.whl

2. 使用 pip 安装下载的 whl

pip install numpy-1.18.1-cp37-cp37m-win_amd64.whl Pillow-7.0.0-cp37-cp37m-win_amd64.whl six-1.14.0-py2.py3-none-any.whl    wxPython-4.0.6-cp37-cp37m-win_amd64.whl

 

3. 测试安装

>>> import wx
>>> wx.version()
'4.0.6 msw (phoenix) wxWidgets 3.0.5'
