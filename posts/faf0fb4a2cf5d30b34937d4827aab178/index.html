[激活官方作弊模式]关于＜野兽与乡巴佬＞中文版游戏的修改-已研究成功

<野兽与乡巴佬>

这篇文章 不是讨论 怎么下载和安装 野兽与乡巴佬,也不是讨论 怎么下载网上的修改器来修改,
而是因为从一开始玩这游戏,在大约十多年前吧,我就想如何能生出一堆的 小孩,以及炸死那些令人讨厌的狼和僵尸.,如果你刚好也是这样,欢迎进来…你可以直接跳过 一二三 ,到四去看

一.这款游戏的魅力

​ <野兽与乡巴佬> 这个游戏可以生孩子,创建自己的家园 ,打怪兽,是一款非常奇葩的游戏

二.下载 方式

​ <野兽与乡巴佬>有中文版和英文版, 先说英文版,下载后在<野兽与乡巴佬>SETUP.EXE上 用兼容模式 选择WIN95 来安装

三.<野兽与乡巴佬> 在WIN10中如何使用

​ 在 此处下载 http://dege.freeweb.hu/dgVoodoo2/dgVoodoo2/
dgVoodoo2不用安装,解压到某个目录 ,先把 Dgvoodoo2 目录 中MS\ x86 拷到乡巴佬目录下,有这么些文件DDraw.dll, D3DImm.dll, D3D9.dll, D3D8.dll

打开 dgVoodooCpl.exe

设置如下:

\17. You should have "dgVoodoo.conf" file in there now

三.窗口化

如果需要窗口化,则把 3DfxSplashDlls.zip 中两个文件,解压缩到windows/system32 和 windows/SysWOW64下
这样 乡巴佬 就可以 按alt+enter 来窗口化和全屏了

四.官方的作弊

前面的在百度都可以找到,不是我们论述的重点
关键的部分如下,以前所有的 <野兽与乡巴佬> 官方的作弊方法 都无法使用,实际上它需要

使用 WinHex 或者其它修改 16进制的程序,打开 <野兽与乡巴佬> 程序目录
BEASTS.EXE #534691处 改为1,必须是英文版的,
改完后 在游戏 里 按F11 ,就可以打开修改控制台

如下图所示,游戏窗口会出现一个CHERT 的字符

作弊方式有两种:

1.输入方式作弊

按F11出现 上图的Edit Script 模式,这个输入模式下,可以通过 输入对应的命令,获取相应的功能,

如输入 ATTRIBUTES (属性)→ 再输入 SPELLS (法术),然后 输入 0 ,再输入1 ,这样就开启了 法师第一项法术
重复这一步 输入1 ,再输入1 ,这样就开启了 法师第二项法术 ,这里 0-6 是wizard(法师)的技能,7-12 则是牧师的技能

ATTRIBUTES (属性) 下面还有 TECHLEVEL 科技水平 ,DISEASE 疾病, NOBUILD 禁止该玩家建造

SETSTATNOW 3 健康 6 生育能力 等等

还可以选择一伙人,然后按S ,再选H ,输入255 ,给他们加血

还可以 Message →createobject→0 →103 → 40,50 //在40,50 坐标处 生出一个小乡巴佬
如果嫌输入 指令太长,也可以 用WinHex 修改BEASTS.EXE #551584 处,把 createobject 改成C,

这样 输入 M→C→0 →103 → 40,50 //就可在40,50 坐标处 生出一个小乡巴佬

还改了D 是删除 , M → D → 删除物品的ID

2.更便捷的作弊方式

输入方式还是复杂了一点,于是我研究出另一种 作弊方式, 按F11,切换 如图所示(Edit Map,Player 0)

然后 输入 o103 ,这时在鼠标所指的位置,就会出现 一个小孩

再按C 键 ,又会出现一个小孩子,你如果重复按C 键,就会出现 越来越多的小孩,

o296则是 轰炸鼠标处,同样 按C 键 可以重复 轰炸,现在可以开始轰炸那些讨厌的狼和僵尸了

o092 可以出来一颗吃人的树

CTRL + F6 则是全屏

同时在下方的建造面板,也会出现一些从未见过的 建筑 或者地图编辑方式

如果建筑不够多,可以参考上面 1.输入方式作弊 中 把科技水平设为最高的5

好了,希望大家 喜欢.
