PHP冒泡排序

冒泡排序

  • 冒泡排序就是比较数组中相邻的两个元素，把较小的元素放到前面，以此类推直到遍历完所有元素
public function the_bubbling(){
    $arr = array(3,6,7,1,9,2,5,4,8);
    for ($i = 0; $i< count($arr); $i++) {
        for ($j = $i+1; $j < count($arr); $j++) {
            if($arr[$i] > $arr[$j]) {
                $temporary = $arr[$i];
                $arr[$i] = $arr[$j];
                $arr[$j] = $temporary;
            }
        }
    }
    echo "<pre>";
    var_dump($arr);
}
