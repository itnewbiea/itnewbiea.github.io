Access Denied when running Spring.IocQuickStart.MovieFinder

I just begin to learn something about Spring.Net, heaps of probem are waiting for me.

 

The very first one has been solved. It is an error saying “Cannot create OBJ folder” when I try to strat the move finder example.

 

This issue could be related to “windows  vista” system only, in vista system, the folders under C:\program folder is not full right even your login in administrator right.

 

I have to right click “C:\Program Files\Spring.NET 1.3.0\examples\”, then select “property”, select “security” to allow “full control” to this folder for my login account.

 

Then problem is solved..
