C语言--编写一个类似库函数strlrn()的函数，求字符串的长度。在main()函数中输入一个字符串，调用该函数后输出结果。要求函数的形参采用指针变量。

源代码：

#include <stdio.h>

int strlen1(char* ps){
	int n=0;
	while(*ps!='\0'){
		n++;
		ps++;
	}
	return n;
}

int main()
{
	char str[30];
	printf("请输入一串字符：");
	gets(str);
	printf("该字符串的长度为%d\n",strlen1(str));
	return 0;
}
