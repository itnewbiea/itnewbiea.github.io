简单的python应用(1)——将文件夹下的图片名称写入txt

简单地将文件夹下的所有图片名称写入txt文件

1.只需要导入os

2.只需要修改paths（储存图片的文件夹路径）和生成的txt文件名称

3.outpath是输出的名字，也可以根据需要来修改

import os
paths = r'./images'  # 储存图片的文件夹路径
f = open('images.txt', 'w')
filenames = os.listdir(paths)  # 读取图片名称
for filename in filenames:
    out = filename
    f.write(out + '\n')
f.close()
