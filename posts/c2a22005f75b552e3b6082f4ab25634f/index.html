js 定时器setTimeout无法调用局部变量的解决办法

   javascript中定时器setTimeout无法调用局部变量，只需要将setTimeout的第一个参数改成函数对象，而不是字符串，就可以了

function rotateloop() {
if (angle < 360) {
angle++;
//use angle
//......
setTimeout("rotateloop()", 100);
}
}

//do something
//......
