LeetCode 560 和为k的子数组

题目描述：

 给定一个整数数组和一个整数 k，你需要找到该数组中和为 k 的连续的子数组的个数。
 ​
 示例 1 :
 ​
 输入:nums = [1,1,1], k = 2
 输出: 2 , [1,1] 与 [1,1] 为两种不同的情况。
 说明 :
 ​
 数组的长度为 [1, 20,000]。
 数组中元素的范围是 [-1000, 1000] ，且整数 k 的范围是 [-1e7, 1e7]。

代码：

 class Solution {
     public int subarraySum(int[] nums, int k) {
         int len = nums.length;
         int count = 0;
         for(int i = 0;i<len;i++){
             int sum = 0;
             for(int j=i;j<len;j++){
                 sum=sum+nums[j];
                 if(sum == k){
                     count++;
                 }
             }
         
         }
         return count;
     }
 }
