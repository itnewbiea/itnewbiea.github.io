常见的Python 2 和 3 差别

Python2.7

print "hello"

hello

Python3.0

print "hello"

  File "tmp.py", line 1
    print "hello"
                ^
SyntaxError: invalid syntax

正确应为：
Python3.0

print("hello")

hello
