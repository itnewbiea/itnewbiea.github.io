搭建TFTP服务器环境搭建

》1.验证tftp服务器环境是否搭建

dpkg -s tftpd-hpa

下图代表安装完成

》2.若是未安装，执行下面命令

sudo apt-get install tftpd-hpa tftp-hpa

》3.修改tftp服务器配置文件

sudo vi /etc/default/tftpd-hpa

修改成如下内容

# /etc/default/tftpd-hpa

TFTP_USERNAME="tftp"
TFTP_DIRECTORY="/tftpboot"
TFTP_ADDRESS=":69"
TFTP_OPTIONS="-c -l -s"

》4.在ubuntu下创建工作目录并设置为最高权限

sudo mkdir /tftpboot

sudo chmod 777 /tftpboot

》5.重启tftp服务

每次打开ubuntu都需要重启此服务

sudo service tftpd-hpa restart

》6.验证是否成功

在服务器工作目录下创建文件

回到家目录

cd ~

登陆tftp服务区

tftp localhost

输入get 1.txt和q

查看验证文件是否到当前目录下
