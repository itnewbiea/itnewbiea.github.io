vue3项目启动报错 in ./src/App.vue?vue&type=style&index=0&lang=scss解决办法


 ERROR  Failed to compile with 1 error                                                                            上午9:18:37

 error  in ./src/views/Login.vue?vue&type=style&index=0&id=26084dc2&lang=less&scoped=true

TypeError: this.getOptions is not a function
第一步 ，先卸载 sass-loader
npm uninstall sass-loader


如果卸载也报错，就执行下的代码
npm uninstall node-sass

npm uninstall sass-loader


第二步 再安装一个sass@1.26.5和sass-loader@7.0.0版本
npm install sass@1.26.5 --save-dev
npm install sass-loader@7.0.0 --save-dev


最后 ，安装低版本的sass-loader需要引入node-sass依赖来支持,请执行下列代码
npm install node-sass
