1、Kubernetes 进入容器方法

---- 方法一：

kubectl exec -it asf-admin-688c7745f-4wk7h  /bin/sh

---- 方法二：

kubectl exec -it nginx-56b8c64cb4-t97vb /bin/bash

2、获取节点相应服务的信息

kubectl get nodes

3、如果需要按selector名来查找相应的pod信息， 可以通过以下命令查看：

kubectl get pod --selector name=tomcat

4、查看K8S集群信息

kubectl cluster-info

5、查看pods所在的运行节点

kubectl get pods -o wide

---- 方法一：

kubectl get pods --all-namespaces

---- 方法二（列出更多的详细信息）：

kubectl get pods -o wide --all-namespaces

7、查看pods定义的详细信息

kubectl get pods -o yaml

8、查看运行的pod的环境变量

kubectl exec <pod名称> env

9、查看指定pod的日志

kubectl logs -f pods/<pod名称> -n kube-system

10、查看集群节点信息

kubectl get nodes
