【尚硅谷云办公问题】Access to XMLHttpRequest at ‘http://localhost:8080/login‘ from origin ‘http://loc

尚硅谷云尚办公P93跨域问题

1、我在后端关于审批的所有Controller都加了@CrossOrigin //跨域注解

2、写了工具类AccessControlAllowOriginFilter

 


//解决跨域问题的方法类
@Configuration
public class AccessControlAllowOriginFilter implements WebMvcConfigurer {

    @Override
    public void addCorsMappings(CorsRegistry registry){
        registry.addMapping("/*/**")
                .allowedHeaders("*")
                .allowedMethods("*")
                .maxAge(1800)
                .allowedOrigins("*");
    }
}
