js将Object类型转为集合类型

JavaScript 中的 Object 类型可以通过调用内置的 Object.values() 或者 Object.entries() 函数将其转换为集合类型。

Object.values() 函数会返回一个数组，数组中包含对象中所有可枚举属性的值。

Object.entries() 函数会返回一个数组，数组中包含对象中所有可枚举属性的键值对。

例如：

constobject = { a: 1, b: 2, c: 3 };

console.log(Object.values(object));  // [1, 2, 3]
console.log(Object.entries(object));  // [['a', 1], ['b', 2], ['c', 3]]

请注意，Object.values() 和 Object.entries() 函数是在 ECMAScript 2017 中引入的，所以可能不在所有浏览器中都可用。
