07:有趣的跳跃

总时间限制: 

1000ms

内存限制: 

65536kB

描述

一个长度为n（n>0）的序列中存在“有趣的跳跃”当前仅当相邻元素的差的绝对值经过排序后正好是从1到(n-1)。例如，1 4 2 3存在“有趣的跳跃”，因为差的绝对值分别为3,2,1。当然，任何只包含单个元素的序列一定存在“有趣的跳跃”。你需要写一个程序判定给定序列是否存在“有趣的跳跃”。

输入

一行，第一个数是n（0 < n < 3000），为序列长度，接下来有n个整数，依次为序列中各元素，各元素的绝对值均不超过1,000,000,000。

输出

一行，若该序列存在“有趣的跳跃”，输出"Jolly"，否则输出"Not jolly"。

样例输入

4 1 4 2 3

样例输出

Jolly

来源

Waterloo local 2000.09.30

 

#include<iostream>
#include<cmath> //abs标准库
#include<algorithm> //sort标准库
using namespace std;

int a[1001],c[1001]; //定义数组，开大点防止溢出 （放在主函数前面定义，值默认为0）

int main() {
int n;
cin>>n;
for(int i=1;i<=n;i++) { 
cin>>a[i];
}
for(int i=1;i<n;i++) {
c[i]=abs(a[i]-a[i+1]); //计算的差的绝对值
}
sort(c+1,c+n); //排序
for(int i=1;i<n;i++) {
if(c[i]!=i) {
cout<<"Not jolly"<<endl; //判断差是否在[1,n-1]之间
return 0;
}
}
cout<<"Jolly"<<endl;
return 0; 
}
