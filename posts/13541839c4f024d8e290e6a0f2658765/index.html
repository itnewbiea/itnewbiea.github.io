vue中安装使用jquery

首先 npm install jquery --save安装依赖文件

然后在 自己的vue.config,js文件中 进行加载 (要是没有就自己新加一个，在最外层)
将jquery导入进去

module.exports = {
  chainWebpack: config => {
    config.plugin('provide').use(webpack.ProvidePlugin, [{
      $: 'jquery',
      jquery: 'jquery',
      jQuery: 'jquery',
      'window.jQuery': 'jquery'
    }])
  }
}

然后在你需要的地方

import $ from 'jquery'
