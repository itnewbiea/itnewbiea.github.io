Jupyter：no module named “xxxx” 但是实际上已安装的解决方案

以安装sympy为例：
在Jupyter中输入

!pip install --upgrade
!pip install sympy 
import sympy

如果不奏效，试试下面这个：

import sys
sys.path.append('my/path/to/module/folder') 
#the (successful) line "!pip install sympy " should tell you where this path is
