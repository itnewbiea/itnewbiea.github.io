C++ sort函数（升序降序排列）

sort 函数类似于快排，执行效率为n*log2(n),优于冒泡排序和选择排序。

#include<iostream>
#include<algorithm>//必须包含的头文件
using namespace std;
int main()
{
	int arr[10] = { 2,4,1,6,9,10,11,23,45,15};
	//升序排列
	sort(arr, arr + 10);//参数分别是起始地址，结束地址，排序分式（默认从小到大）
	for (int i = 0; i < 10; i++)
		cout << arr[i] << " ";
	cout << endl;
	//降序排列
	sort(arr, arr + 10, greater<int>());//可以使用模板，也可以自己写比较函数
	for (int i = 0; i < 10; i++)
		cout << arr[i] << " ";
}
