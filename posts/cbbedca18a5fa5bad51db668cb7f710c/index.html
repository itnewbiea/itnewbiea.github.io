js中创建对象、集合

有时候有些数据需要在内存中完成，等用户确认后才能持久化，这个时候就需要在前台页面中创建对象、集合，保存用户的操作

js中创建集合

var list=[];

js中创建cs对象

var cs = {

    "id"：1,

    "name":'admin',

    "password":'admin'

}

保存对象

list.push( cs );
