背景

因项目环境需要，欲将Ubuntu18.04升级至Ubuntu20.04，参考网上其他小伙伴的方法，也遇到了一个问题，特此记录一下，希望能帮助其他有同样问题的小伙伴。

升级方法

参考：第十五章 Ubuntu18.04LTS升级到20.04LTS
主要的步骤：

sudo apt update
sudo apt upgrade
sudo apt install update-manager-core
sudo do-release-upgrade

遇到的问题

在执行“do-release-upgrade”时，遇到“Failed to connect to https://changelogs.ubuntu.com/meta-release-lts. Check your Internet connection or proxy settings”的问题。
