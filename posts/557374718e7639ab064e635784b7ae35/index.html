sql中可以使用不在select中的字段排序

案例: 电梯最大载重量为 1000; 最后一个能进入电梯的人

select 
  person_name
from(
  select 
    person_name,
    sum(weight) over(order by turn) t
  from Queue
)t1
where 
  t<=1000
order by t desc
limit 1;

在hive中也是可以的

hive > select * from stu;

stu.id	stu.name

3	cc

hive > select name from stu order by id desc;
