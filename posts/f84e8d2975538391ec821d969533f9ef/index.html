docker 常见命令学习

  1. 获取镜像 docker pull ubuntu
  2. 启动容器 docker run -it ubuntu /bin/bash [ i 交互式操作 t 终端]
  3. 后台运行容器 docker run -itd ubuntu /bin/bash
  4. 进入后台运行的容器
    1. docker attack <容器id> [ 该方法退出会导致容器停止 ]
    2. docker exec <容器id> [ 该方法退出不会导致容器停止]
  5. 启动一个停止的容器 docker start <容器id>
  6. 查看容器 docker ps
  7. 导出容器 docker export <容器id> > ubuntu.tar
  8. 导入容器 cat ubuntu.tar | docker import - test/ubuntu:v1
  9. 删除容器 docker rm -f <容器id>
  10. 查看容器端口 docker port <容器id>
  1. 查看本地镜像列表 docker images
  2. 查找镜像 https://hub.docker.com 或 docker search httpd
  3. 删除镜像 docker rmi <镜像>
  4. 更新镜像 docker commit -m="update" -a="author" <容器id> <新镜像名称：TAG> [ m 提交的描述信息 a 镜像作者]
  5. 构建镜像（需要使用Dockerfile） docker build -t dockername . [ t 指定要创建的目标镜像名称]
  6. 为镜像添加一个新标签 docker tag <标签> <image>
  1. 创建容器时命名 docker run --name myname ubuntu /bin/bash
  2. 新建docker网络 docker network create -d bridge <net-name>
  3. 连接容器 docker run -itd --name t1 --network <net-name> ubuntu /bin/bash
