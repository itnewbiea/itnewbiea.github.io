关于两个double类型数的大小判断

大于小于是可以像整数那样直接判断的，但是不可以判断两个double是相等的。

因为

double类型的表示往往是不精确的(近似表示). 有精度范围的那种. 如果是运算结果,即使都是0.0,也不保证相等!一般用一个容差e(很小的值)
那么a和b相等就可以表示为

if ( fabs(a-b) < e )
return true;

注意：double类型虽然可以通过＞和＜直接比较，但是大于等于，小于等于是不能直接>=和<=比较的。
错误比较方式：now >= mid*n
正确比较方式：now - mid * n > -0.000001
