Debian11 安装Chromium浏览器

1 sudo apt-get update

2 sudo apt-get install chromium chromium-driver

3 安装语言包，中文

sudo apt-ger install chromium-l10n

4 查看版本

chromium --version

其他：也可以直接从 新立得软件中心直接搜索 chromium 安装。
