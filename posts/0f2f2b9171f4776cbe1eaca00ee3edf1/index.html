js对一个对象进行存取key和value值操作

javascript中对一个对象: var obj = {}
进行存储操作：

// 前面为key值, 后面为value值
obj[key] = value;

进行取出操作：

// 打印的key为key值, obj[key]为该key对应的value值
for(var key in obj){
  console.log('key is ' + key +' and value is' + obj[key]);
}
