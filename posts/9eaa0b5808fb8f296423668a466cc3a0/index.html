问题描述

今天写了一个对象方式传参的get请求接口方法，发现没有载荷，ip地址也没有带查询字符串，数据也没有响应。

错误分析

实际上这里的query是对象方式带参跳转的参数名，而get方法对象方式传参的参数名是parmas

解决方法

get请求有两种带参的方式

  • 方式1：对象形式传参，参数名是params

  • 方式2：查询字符串带参，添加到地址?后面
       export function delUders(id){
           return request({
               url:'/system/users/list?'+id,
               method:'get'
           })
       }
