windows中搭建maven环境

1，下载maven，下载地址 http://maven.apache.org/download.cgi，点击 apache-maven-*.*.*-bin.zip 进行下载

2，解压到对应目录中

3，配置环境变量：

1

2

M2_HOME = E:\Program Files\Java\apache-maven-3.5.0

path = .....;%M2_HOME%\bin

4，测试安装，打开cmd，输入 mvn -v

5，自定义Maven的本地库路径，可以去修改Maven的配置文件，位置在： 你的路径\apache-maven-*.*.*\conf\setting.xml，如下：

<!-- localRepository

   | The path to the local repository maven will use to store artifacts.

   |

   | Default: ${user.home}/.m2/repository

  <localRepository>E:/Program Files/Java/repo</localRepository>

  -->

<localRepository>E:/Program Files/Java/repo</localRepository>

默认是在 ${user.home}/.m2/repository ，我们改为了E:/Program Files/Java/repo

6，用一个命令验证一下：

打开命令提示符，输入：mvn help:system 

该命令会打印出所有的java系统属性和环境变量。这些信息对我们日常的编程工作很有帮且。

如果运行的过程中没有错误，打开我们仓库（E:/Program Files/Java/repo）会发现里面多了一些文件。这些文件就是我们从maven的中央仓库下载到本地仓库的。
