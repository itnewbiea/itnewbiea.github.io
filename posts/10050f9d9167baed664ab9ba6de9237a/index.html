提示：本文采用源码方式安装，原因是操控灵活，可以自定义增删模块；其他的方式想要实现增删模块操作不方便；

    • 2.1 执行yum -y install pcre pcre-devel

    • 2.3 执行 yum install -y openssl openssl-devel
  • 3. 安装Nginx
    • 3.1 下载Nginx安装包
    • 3.2 解压(打印内容太多没展示)
    • 3.3 进入安装目录
    • 3.4 执行自动配置
    • 3.5 编译并安装

      • 3.6.1 进入环境变量配置文件

      • 3.6.3 使环境变量生效
      • 3.6.4 启动Nginx
      • 3.6.5 检验Nginx是否启动成功
