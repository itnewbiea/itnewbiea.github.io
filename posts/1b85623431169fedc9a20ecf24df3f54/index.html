electron

concurrently 使用concurrently这个npm包可以同时执行两个命令

"dev": "concurrently \"electron .\" \" npm start\"",

https://www.npmjs.com/package/wait-on 等一个命令执行完成后再执行另一个命令
 

"dev": "concurrently \"wait-on http://localhost:3000 && electron .\" \"npm start\"",

cross-env 解决跨平台设置环境变量的问题。但是这次我们使用它是为了利用它的 BROWSER=none 来解决上面提到的问题，不打开浏览器中的 tab 页。
 

 "dev": "concurrently \"wait-on http://localhost:3000 && electron .\" \"cross-env BROWSER=none npm start\"",

消息

在渲染进程中显示消息通知

const myNotification = new Notification('Title', {
  body: 'Notification from the Renderer process' // 通知的内容
})

myNotification.onclick = () => {
	// 点击之后执行
  console.log('Notification clicked')
}

在主进程中显示

const { Notification } = require('electron')

function showNotification ()
  const notification = {
    title: 'Basic Notification',
    body: 'Notification from the Main process'
  }
  new Notification(notification).show()
}

app.whenReady().then(createWindow).then(showNotification)

https://www.electronjs.org/docs/tutorial/notifications
