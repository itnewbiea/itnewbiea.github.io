C语言：求e的值。 e=1+1/1! +1/2! +1/3! ...+1/n!（for循环）

用for循环，计算前50项。

方法:应用for循环

#include<stdio.h>
void main()
{
    int i,n;
    double sum =1,t=1;
    printf("输入n:\n");
    scanf("%d",&n);
    for(i=1;i<=n;i++)
    {
        t=t*i;
        sum= sum+1/t;
    }
    printf("e=%lf",sum);
}
