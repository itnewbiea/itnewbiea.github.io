使用过的moment对象的用法（至2022/11/03）

1. 初始化一个moment对象

moment(String|Moment|Date, format)
// e.g.
moment('19', 'HH')
moment(new Date(), 'yyyy-MM-dd')
// 字符串转moment对象
moment('2022-11-3 13:12:23', 'YYYY-MM-DD HH:mm:ss')

2. moment对象转字符串(格式化)

moment().format()
moment().format('YYYY-MM-DD')

3. moment对象转时间戳

moment().valueOf()

4. moment对象的取值

moment.year()  // 年份
moment.month()  // 月份
moment.date()  // 日
moment.hour()  // 小时
moment.minute()  // 分钟
moment.second()  // 秒

5. 比较两个moment对象是否相同

moment.isSame(Moment,String)
// e.g.
moment('2010-10-20').isSame('2009-12-31', 'year');  // false
moment('2010-01-01').isSame('2010-02-01', 'day');   // false, 不同的月份
moment('2010-01-01').isSame('2010-01-01', 'day');   // true

6. 今天之后的日期

// 禁用当天之后的日期
disabledDate(current) {
	return current && current > moment().endOf('day')
},

7. 当前时间往前推十个小时

moment().subtract(10, 'hours')
