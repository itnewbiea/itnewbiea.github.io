SpringBoot项目初始化配置（Spring Initializr）

  • Group：即GroupID,表示项目组织的标识符，实际对应Java的包结构，是main目录里Java的目录结构。
  • Artifact：即ArtifactId，表示项目的标识符，实际对应项目的名称，也就是项目根目录的名称。
  • Description：表示项目描述信息。
  • Package name：表示项目包名。

生成项目的pom.xml文件中显示内容如下，对应上面的Group,Artifact,Version,Name,Description

    <groupId>com.example</groupId>
    <artifactId>demo</artifactId>
    <version>0.0.1-SNAPSHOT</version>
    <name>demo</name>
    <description>Demo project for Spring Boot</description>

生成的目录结构：

可以看到项目的根目录名称为：demo,对应 Artifact

项目的包名称为：com.example.demo,对应Package name

com.example对应GroupId
