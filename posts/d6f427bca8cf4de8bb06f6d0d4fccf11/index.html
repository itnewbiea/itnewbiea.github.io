Ant Design Pro V4 安装

安装

yarn create umi

或则使用npm

npm create umi

然后选择：ant-design-pro

其他的分别是：

app  — 用简单的样板创建项目，支持typescript
block  —  创建一个 umi 块
library —  用umi.创建一个库
plugin —  创建 umi 插件

选择版本:Pro V4

选择语言：js或则ts

选择完整版:complete
