C语言 统计数字字符的个数

Description

输入一行字符，统计出其中数字字符的个数。

Input

一行字符串，总长度不超过255。

Output

输出为1行，输出字符串里面数字字符的个数。

Sample Input

Peking University is set up at 1898.

Sample Output

4

Source Code

#include<stdio.h>
#include<string.h>
int main(void)
{
	int count=0;
	char a[1000];
	gets(a);
	for(int i=0;i<strlen(a);i++)
	{
		if(a[i]>='0'&&a[i]<='9')
			count++;
	}
	printf("%d\n",count);
	return 0;
}
