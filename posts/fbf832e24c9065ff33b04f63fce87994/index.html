JAVA获取磁盘名称

代码

FileSystemView fileSystemView = FileSystemView.getFileSystemView();
File[] files = File.listRoots();
for (File file : files) {
    System.out.println(fileSystemView.getSystemDisplayName(file));
}

输出

Windows (C:)
软件 (D:)
数据 (E:)
