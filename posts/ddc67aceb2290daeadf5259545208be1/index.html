Mac下命令行下 报 env: bash\r: No such file or directory

我是从window下拷贝的 react-native 项目执行 react-native run-android 时候报 env: bash\r: No such file or directory
解决办法：

1、brew install dos2unix
2、find . -type f -exec dos2unix {} \;

第二步的分号前有个\
