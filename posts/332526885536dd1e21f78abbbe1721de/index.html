JavaScript：for循环之1~100的加法运算

问题：求 1~100 之间的整数累加和

首先我们先分析一下这个问题的运算思路。

思路分析：

1~100需要循环100次

需要一个计数器  i

需要一个存储结果的变量sum，初始值是 0

算法：1 + 2 + 3 + 4 + 5+......+ 100 , sum = sum + i

算法分析：

 代码：

var sum = 0;    // 求和，变量
for (var i = 1; i <=100; i++) {
    sum = sum +i;    // 另一种写法：sum += i;
}
console.log(sum);
