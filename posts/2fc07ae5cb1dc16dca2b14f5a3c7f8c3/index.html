Only digits (0-9) can be put inside [] in the path string: daijiaoList[${index}].number

单独修改某一项的值的时候出现这样的错误 代码是这样的

 var up = "daijiaoList[${index}].number"

 this.setData({
   [up]:10
 })

这时候代码才报错 

修改一下后

this.setData({[`daijiaoList[${index}].number`]:10})

这样就可以了
