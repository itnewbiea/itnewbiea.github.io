c语言中fclose函数的使用你会吗

点击上方蓝字关注我，了解更多咨询

1、C语言fclose函数用于关闭使用fopen成功打开的文件。

2、fopen函数和fclose函数总是成对出现。如果文件成功打开fopen函数，请使用fclose函数关闭文件。

实例

#include <stdio.h>
#include <stdlib.h>
 
int main()
{
   //如果D:/file.txt存在，则fopen打开成功，否则失败返回NULL
   FILE * fp = fopen ("d://file.txt", "r");
   if(!fp)
       printf("打开文件失败");
   else
   {
       printf("打开文件成功")；
       fclose(fp); //如果文件打开成功记得关闭
   }
   return(0);
}

END
