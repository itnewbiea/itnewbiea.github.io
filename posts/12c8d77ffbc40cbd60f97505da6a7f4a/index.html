vue cli更换版本

vue-cli更换版本

vue更换版本

最近要写个小项目，用到了vue，于是安装了vue cli，但是在使用过程中发现最新的版本好像不太好用，想换成旧版本。

查看当前版本

通过vue -V可以查看当前版本，现有版本是5.0.8，如图所示。


然后卸载掉当前版本，使用命令npm uninstall -g @vue/cli即可。

安装新版本

然后使用命令npm i -g @vue/cli@5.0.4安装想要更换的版本即可。其中，第二个@后面跟的是版本号。

涉及的命令如下，可以直接复制

1.下载当前版本

npm uninstall -g @vue/cli

2.安装新版本

npm install -g @vue/cli@版本号

3.查看当前版本号

vue -V
