Java中a++和++a的区别

//首先a++和++a都是a=a+1的简写过程，只是方便，所以这么写，在后面for循环里要经常用到a++

public class Demo01{

public static void main(String[] args){

int m = 7;

int n = 7;

int x = 2*m++;

int y = 2*++n;

System.out.println(x+" "+y);// x输出 14 y，输出16,m和n都是8，通过运行结果可以看出，m++是先将m=7的值运算后 ，m才+1。x=2*7，m=7+1。而++n就是先将n=7+1=8算出后在进行赋值。y=2*（7+1）；

}

}
