LDR2001电脑免驱USB转串口芯片方案

USB转串口对于需要用到串口的用户们提供了便利，下面介绍的这款LDR2001芯片功能强大，电脑免驱，自带Billboard功能。

LDR2001 芯片是乐得瑞科技设计的一个免驱的 USB 转串口的 CDC 类芯片，可以实现USB 转串口和 Billboard 的功能。当使用了 LDR2001 芯片的设备接入 USB HOST 时，会在HOST 端弹出 Billboard 窗口，提醒用户该设备所具有的 Alternate Mode 功能。电脑直接免驱动使用。

LDR2001特点：

  1. 全速 USB 设备接口，支持 1.1 以上版本，Billboard 支持 USB2.01 以上的版本，外围元器件无需晶体和电容。

  2. 仿真标准免驱串口，用于升级原串口外围设备，或者通过 USB 增加而外串口。

  3. 计算机端无需下载驱动，Billboard 设备类在 Windows10 以上系统免驱。

  4. 硬件全双工串口，内置收发缓冲区，支持通讯波特率 50bps～2Mbps。

  5. 无奇偶，偶数，奇数，标记或空间奇偶校验。

  6. 支持奇偶校验，溢出和成帧错误。

  7. 支持单通道 RS-232 和 RS-422 接口。

  8. 支持 5V 电源电压和 3.3V 电源电压。

  9. 支持本地 Billboard 设备类驱动程序。

 

LDR2001采用MSOP10封装，体积小

功能说明

LDR2001 芯片内置了 USB 上拉电阻，UDP 和 UDM 引脚应该直接连接到 USB 总线上。

LDR2001 芯片内置了电源上电复位电路。还提供了低电平有效的外部复位输入引脚。

LDR2001 芯片已经内置时钟发生器，无需外部晶体及震荡电容。

LDR2001 芯片支持 5V 电源电压或者 3.3V 电源电压。当使用 5V 工作电压时，

LDR2001 芯片的 VCC 引脚输入外部 5V 电源，并且 V3 引脚应该外接容量为 0.1uF 的电

源退耦电容。当使用 3.3V 工作电压时，LDR2001 芯片的 V3 引脚应该与 VCC 引脚相连

接，同时输入外部的 3.3V 电源，并且与 LDR2001 芯片相连接的其它电路的工作电压不

能超过 3.3V。

LDR2001 内置了独立的收发缓冲区，支持单工、半双工或者全双工异步串行通讯。串

行数据包括 1 个低电平起始位、5、6、7 或 8 个数据位、1 个或 2 个高电平停止位，支

持奇校验/偶校验/标志校验/空白校验。LDR2001 支持常用通讯波特率：50、75、100、

110、134.5、150、300、600、900、1200、1800、2400、3600、4800、9600、14400、

19200、28800、33600、38400、56000、57600、76800、115200、128000、153600、

230400、460800、921600、1500000、2000000 等。

LDR2001 芯片串口发送信号的波特率误差小于 1％。

在计算机端的 Windows 操作系统下，LDR2001 是一个免驱的芯片，能够仿真标准串口，

所以绝大部分原串口应用程序完全兼容，通常不需要作任何修改。

LDR2001 可以用于升级原串口外围设备，或者通过 USB 总线为计算机增加额外串口。

通过外加电平转换器件，可以进一步提供 RS232、RS485、RS422 等接口。

LDR2001 可以配置 Billboard 描述符，根据弹出窗口的不同知道通信的问题所在。
