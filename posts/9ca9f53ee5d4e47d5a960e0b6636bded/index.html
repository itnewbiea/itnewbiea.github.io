用户回答回答于 2018-08-23

你可以为电子邮件字符串创建一个简单的包装类：

public class EmailAddress {

    @Pattern("\b[A-Z0-9._%+-]+@[A-Z0-9.-]+.[A-Z]{2,4}\b.")
    String email;

    //getters and setters
}

然后标记@Valid现有对象中的字段：

@NotNull
@Valid
List<EmailAddress> emailIds;

然后验证器将验证列表中的每个对象。
