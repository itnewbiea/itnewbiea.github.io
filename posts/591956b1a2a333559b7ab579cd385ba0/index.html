【小程序开发】解决 HBuilder X “[error] IDE service port disabled“无法运行问题

        开发微信小程序时，在HBuilder X上方菜单点击”运行“->”运行到小程序模拟器“->”微信开发者工具“后，控制台输出了”[error] IDE service port disabled. To use CLI Call, please enter y to confirm enabling CLI capability, or manually open IDE -> Settings -> Security Settings, and set Service Port On.“ ... ? Enable IDE Service (y/N) ESC[27DESC[27C，如下图：

        说明微信开发者工具的服务端口是关闭状态，根据提示，第一种在控制台直接输入”Y“的方式好像行不通，那么就使用第二种手动打开工具 -> 设置 -> 安全设置，将服务端口开启。注意：这个设置是在微信开发者工具中的，而不是HBuilder X，如下图：
