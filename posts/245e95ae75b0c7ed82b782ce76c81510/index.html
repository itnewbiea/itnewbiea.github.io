python 2个列表相减的2种写法

list1 = [1, 2, 3, 4, 5, 6]
list2 = [1, 3, 4, 5]
第一种
list3 = [item for item in list1 if item not in set(list2)]
print(list3)  # [2, 6]
第二种
print(set(list1) - set(list2))  # {2, 6}
