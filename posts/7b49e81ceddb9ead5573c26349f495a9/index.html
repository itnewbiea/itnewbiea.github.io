点击input时，去除边框


方法

代码展示：

       input {
            border: none;
            -webkit-box-shadow: none;
            box-shadow: none;
            outline: none;
        }

        input:focus {
            border: none;
            box-shadow: none;
        }
  1. outline样式属性：outline属性表示轮廓具体位置如下图所示（位于经典盒子模型的border和margin之间）：
    在默认状态下，对处于focus状态的元素浏览器会通过虚框或者外发光的形式进行区分和提示，这个虚框或者外发光就是浏览默认的outline。这个outline是很有用的，它可以提示用户自己当前聚焦在哪个元素上面，而且在此时按下回车键就相当于点击了这个元素。

2.:focus 伪类选择器：用于选取获得焦点的表单元素，焦点也即是光标。
