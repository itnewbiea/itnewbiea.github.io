const与var的比较

var声明了一个变量，并且可以同时初始化该变量。
let语句声明一个块级作用域的本地变量，并且可选的赋予初始值。
const 声明创建一个只读的常量，作用域与let相同。这不意味着常量指向的值不可变，而是变量标识符的值只能赋值一次。

 

  1. const和var的明显区别是，const声明的是常量，不可被后面的代码赋值改变：
  2.  
  3. var x = 1;
  4. x = 2;
  5. console.log(x);//2;-----var声明的是变量，可被赋值替换
  6.  
  7. const y = 1;
  8. y = 2;
  9. console.log(y);//Assignment to constant variable.;------const声明的是常量，不可改变
