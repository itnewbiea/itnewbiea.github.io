C语言--编写程序，从键盘输入月份号，输出该月的英文名。（用指针知识）

源代码：

#include <stdio.h>

int main()
{
    char *month[]={"January", "Februray", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"};
    int m;
    printf("请输入月份号：");
    scanf("%d", &m);
    if (m >= 1 && m <= 12)
		printf("%s\n",*(month+m-1));
	else
		printf("输入错误\n");
    return 0;
}
