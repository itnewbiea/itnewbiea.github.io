错误 C2733: 不允许重载函数的第二个 C 链接


头文件声明
#ifdef __cplusplus
extern “C” {
#endif
#ifdef __cplusplus
}
#endif
没有参数错误选项，最终解决。
解决办法：
你的包含头文件中可能存在非C文件，例如#include<string>,会引起该错误，改为#include<string.h>。
