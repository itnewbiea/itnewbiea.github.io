python tcping（ping命令)

使用tcping库中的ping方法使用如下：

from tcping import Ping
ping = Ping('www.baidu.com', 80, 200)  # 地址、端口、超时时间
ping.ping(3)  # ping命令执行次数
ret = ping.result.table  # 以表格形式展现（ping.result.raw  # 原始形态，ping.result.rows  # 行显示）
print(ret)

其他：
该库中其他方法产生的值可以以字典的形式展现，

tcping.Ping('www.baidu.com', 80, 10).__dict__  # .__dict__

奇技淫巧

—通过修改tcping库参数，让ping间隔发生改变
