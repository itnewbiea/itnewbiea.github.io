Ubuntu18下安装PCL库

前提：
VTK-已安装(sudo apt-get install libvtk7.1-qt libvtk7.1 libvtk7-qt-dev)
cmake version 3.10.2 已安装

1、下载PCL

git clone https://github.com/PointCloudLibrary/pcl.git

2、进入目录编译文件

cd pcl
mkdir build && cd build
cmake ..
make -j8

3、编译后成功

sudo make install


至此，PCL安装成功！
