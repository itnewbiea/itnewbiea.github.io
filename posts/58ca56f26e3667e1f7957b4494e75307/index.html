完整提交流程 https://www.ieee-ras.org/publications/ra-l/submission-procedures

1 RAL注册

https://ras.papercept.net/journals/ral

  1. 先注册获得一个PIN码（填完信息之后，会出现一个名单，这个名单与你填的信息有部分相同，目的是确认自己有没有注册过）
  2. 再设置密码（验证码会发到注册设置的邮箱）。
  3. 登陆后，要补充个人信息。
  4. RAL对提交论文的说明（这个页面就在登录进去的那个页面，或者home界面点PIN选项卡）

2 ORCID ID 绑定RAL账号PIN码

ORCID ID 即 Open Researcher and Contributor ID 什么是orcid？

  1. 注册 https://orcid.org/ 。
  2. 关联。也就是上述图片的最后6条。链接在RAL登录界面，然后点击授权。然后会回到一个RAS的登录界面，点击登录，即完成关联。

3 提交

  1. 提交链接位置http://ras.papercept.net/journals/ral/scripts/login.pl

  2. 提交步骤

  3. 注意

    1. 作者，一作，通信等确认
    2. 关键字，最少可以输入 2 个，最多 5 个（推荐）关键字；前两个关键字必须从RA-L 学科领域列表中选择；最多可自由输入三个关键词，（实际上，只能输入三个列表里的关键字，然后按关键字分配审稿人）
    3. 写一个cover note，内含对主编说的话，指出创新点，→模板
    4. IEEE投稿报错解决方法“Page n has margin impositions”
    在\maketitle后面加上
\pagestyle{empty}  % no page number for the second and the later pages
\thispagestyle{empty} % no page number for the first page
