Android Studio的LogCat过滤栏(选择设备界面)不见了

打开LogCat界面方法：

1、View --- Tool Windows --- Event Log

 

2、在 android studio整个界面的左下角，鼠标移动过去会自动弹窗界面

 

LogCat的过滤栏界面

 有时候会莫名其妙的不见了这个过滤栏

解决方式：

1、尝试移动log框到左右上下（效果最好）

2、尝试启用并停用 Split Mode （有时有效）

3、尝试启用并停用 Windowed Mode （好像比较靠谱）

4、尝试重启 Studio （低概率修好）

5、尝试重启电脑 （有时有效）
