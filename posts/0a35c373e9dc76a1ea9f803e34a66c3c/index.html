split()分割字符串

第一种分割：

String str = "abacad"
String[] ary = str.split("a");
得到的ary是{"b", "c", "d"}，长度为3
b
c
d

第二种分割：

String str = "abacad"
String[] ary = str.split("[a]");
得到的ary是{"", "b", "", "c", "", "d"},长度为5

b

c

d
加了中括号的分隔符"[]"会将相同符号转为空字符""
