01补充一下：哈希表的基础知识

1. 创建哈希表

在 JavaScript 中，您可以使用对象或 Map 来创建一个哈希表。

  • 使用对象：
let hashTable = {};
  • 使用 Map 对象：
let hashMap = new Map();

2. 插入键值对（Add / Set）

向哈希表中添加键值对。

  • 使用对象：
hashTable[key] = value;
  • 使用 Map 对象：
hashMap.set(key, value);

3. 搜索键值（Get）

根据键获取哈希表中的值。

  • 使用对象：
let value = hashTable[key];
  • 使用 Map 对象：
let value = hashMap.get(key);

4. 检查键是否存在（Check for Key）

检查哈希表中是否存在某个键。

  • 使用对象：
let exists = key in hashTable;
  • 使用 Map 对象：
let exists = hashMap.has(key);

5. 删除键值对（Delete）

从哈希表中删除一个键值对。

  • 使用对象：
delete hashTable[key];
  • 使用 Map 对象：
hashMap.delete(key);

6. 遍历哈希表（Iterate）

遍历哈希表中的所有键值对。

  • 使用对象：
for (let key in hashTable) {
    if (hashTable.hasOwnProperty(key)) {
        console.log(key, hashTable[key]);
    }
}
  • 使用 Map 对象
hashMap.forEach((value, key) => {
    console.log(key, value);
});

7. 获取哈希表大小（Size）

获取哈希表中的键值对数量。

  • 使用对象：需要手动计算。
let size = Object.keys(hashTable).length;
  • 使用 Map 对象：
let size = hashMap.size;

02JS常见数据结构

在 JavaScript 中，数组（Array）、集合（Set）和映射（Map）都是用于存储数据的常见数据结构，但它们各自有不同的特性和用途：

数组（Array）

  1. 有序集合：数组是元素的有序集合，其中每个元素都有一个数字索引。

  2. 重复元素：数组可以包含重复的元素。

  3. 元素类型：数组中的元素可以是任意类型，包括不同类型的混合。

  4. 使用场景：适用于需要顺序访问或快速访问特定索引元素的场景。

  5. 基本操作：添加、删除、访问元素等，以及一系列数组特有的方法（如 map、filter、reduce 等）。

集合（Set）

  1. 无序集合：Set 是无序的集合，不保证元素的顺序。

  2. 唯一元素：Set 中的元素是唯一的，不允许重复。

  3. 元素类型：Set 中的元素可以是任意类型。

  4. 使用场景：适用于需要存储不重复元素、快速检查元素是否存在的场景。

  5. 基本操作：添加、删除元素，以及检查元素是否存在。

映射（Map）

  1. 键值对：Map 存储键值对集合，其中键和值可以是任意类型。

  2. 有序集合：Map 保持键值对的插入顺序。

  3. 唯一键：Map 中的键是唯一的。

  4. 使用场景：适用于需要快速访问、更新或删除键值对的场景。

  5. 基本操作：添加、删除键值对，访问或更新值，以及遍历键值对。

总结

  • 数组（Array）：有序集合，可包含重复元素，通过数字索引访问元素。
  • 集合（Set）：无序集合，元素唯一，不包含重复元素，常用于去重和快速查找。
  • 映射（Map）：键值对集合，保持插入顺序，键唯一，适用于存储和处理相关联的数据。

    • seen 集合为空。

  2. 第一次循环：

    • 19 不等于 1，且 19 不在 seen 集合中，因此继续执行。
    • 将 19 添加到 seen 集合中。
    • 计算 19 的各位数的平方和：1*1 + 9*9 = 1 + 81 = 82。
    • 更新 n = 82。

  3. 第二次循环：

    • 82 不等于 1，且 82 不在 seen 集合中。
    • 将 82 添加到 seen 集合中。
    • 计算 82 的各位数的平方和：8*8 + 2*2 = 64 + 4 = 68。
    • 更新 n = 68。

  4. 后续几次循环：

    • 类似地，计算 68 的各位数平方和得到 100，然后计算 100 的各位数平方和得到 1。
    • 更新 n = 1。

  5. 结束循环：

    • n 等于 1，跳出循环。

  6. 返回结果：

    • n === 1 为 true，因此函数返回 true，表示 19 是快乐数。
