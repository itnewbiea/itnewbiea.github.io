【题目描述】

AB两人石头剪刀布，0是石头，1是剪刀，2是布。赢一局加两分，平局各加一分。

【输入描述】

读入文件game.in

一个正整数n(n<=100)

以下n行分别是AB两人出的是什么

【输出描述】

输出文件game.out

两个正整数分别是AB两人的分数

【样例输入】

3
0 1
2 2
0 2

Copy

###【样例输出】

3 3

#include<bits/stdc++.h>
using namespace std;
int A,B,C=0,D=0,n;
int main(){
    freopen("game.in","r",stdin);
    freopen("game.out","w",stdout);
    cin>>n;
    while(n>0){
        cin>>A>>B;
        if((A==0&&B==1)||(A==1&&B==2)||(A==2&&B==0)){
            C=C+2;
        }
        else{
        if(A==B){
            C++;
            D++;
        }
    else{
        D=D+2;
            }
        }
        n--;
    }
    cout<<C<<' '<<D;
    fclose(stdin);fclose(stdout);
    return 0;
}
