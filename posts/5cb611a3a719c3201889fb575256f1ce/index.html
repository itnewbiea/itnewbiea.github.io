Cydia无法联网？ 

方式1. 

iphone5s 以上下载 乐网 ,根据教程操作。（https://www.i4.cn/news_detail_18849.html） 

 

方式2. 

Cydia 不能联网的终极解决方法。（https://blog.csdn.net/sysprogram/article/details/86558683）

首先，安装 MobileTerminal (手机终端)，根据教程制作ipa安装包。（参考文章）

具体操作如下（windows步骤）：

(1)下载 CydiaImpactor、MobileTerminal.deb、爱思助手、7zip、winrar。

(2) 使用 7zip 打开 MobileTerminal.deb，点击提取将文件提取出来。

(3) 提取的文件里有 data.tar，解压之后，里面有一个 Applications 目录，进去后会看到 MobileTerminal.app，新建 Payload 目录，将 MobileTerminal.app 放入 Payload 目录里，然后选择 Payload 打包成 zip 格式，重命名为 MobileTerminal.ipa

然后，在手机上安装ipa。

具体步骤（接上述）：

(4) 打开 CydiaImpactor，将 MobileTerminal.ipa 拖到 CydiaImpactor 上，输入你的 Apple ID 账号和密码，将 MobileTerminal 安装到手机上。

(5) 打开爱思助手，将准备好的 lib 文件上传到 books 目录

(6) 在手机上打开 MobileTerminal，输入以下命令，这样就解决了。密码（alpine）

su

cp -R /var/mobile/Media/Books/lib /var

遇到如下错误情况： 

error1：

Please sign in with an app-specific password. You can create one at appleid.apple.com.

reason1：账号开启双重验证，需要生成特殊的App专用密码。

solve1：https://blog.csdn.net/mqf163/article/details/53395746

 

error2:

 The user name or passphrase you entered is not correct

reason2:Mac上钥匙串App的login未上锁，需要锁住。

solve2:https://github.com/desktop/desktop/issues/3625

 

error3:

iOS Failed to find matching arch for 64-bit Mach-O input file

slove3:https://blog.csdn.net/LorinRain/article/details/51916594

大概是，iphone5的处理器是armv7 和 而ipa适用于arm64处理器 

 

error4:PLIST_NONE not PLIST_STRING

slove4:http://bbs.iosre.com/t/iphone6s-10-3-1-plist-hpp-201-not-plist-string/11207

emmmm…...多尝试吧……虽然一直报这个错误，意外的是打开手机发现Terminal已经装好了…… 

 

至此，Terminal终于安装完成，执行删除文件操作。（密码：alpine） 

su

cd /var/preferences

rm com.apple.networkextension.plist

rm com.apple.networkextension.cache.plist

rm com.apple.networkextension.necp.plist

 

最后，重新打开Cydia，即可加载页面。
