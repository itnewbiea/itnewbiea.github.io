java.lang.IllegalArgumentException: object is not an instance of declaring class

java反射的时候报错。

原因是因为没有实例化一个类

	setMethod.invoke(t, new Object[]{columnValue});

这一行报错。

应该继承泛型类的空构造方法

setMethod.invoke(clazz.newInstance(), new Object[]{columnValue});

OK，这样就解决了。
