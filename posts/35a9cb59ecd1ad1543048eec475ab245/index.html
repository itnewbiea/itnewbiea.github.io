在ubuntu中，如何开机自启动一个有界面程序

有多种方法可以开机自启动一个有界面程序，以下是其中两种常用的方法：

方法一：使用gnome-session-properties

  1. 打开终端，输入命令：gnome-session-properties，打开“启动应用程序首选项”窗口。

  2. 点击“添加”按钮，填写应用程序的名称、命令和注释，例如：

    名称：My Program
    命令：/usr/bin/myprogram
    注释：启动My Program

  3. 点击“添加”按钮后，该应用程序会被添加到“启动应用程序”列表中。

  4. 关闭窗口，重启电脑后该应用程序会自动启动。

方法二：使用rc.local

  1. 打开终端，输入命令：sudo nano /etc/rc.local，编辑rc.local文件。

  2. 在文件末尾添加应用程序的启动命令，例如：

    /usr/bin/myprogram &

  3. 保存文件并退出。

  4. 重启电脑后，该应用程序会自动启动。

注意：使用rc.local方法需要注意以下几点：

  • rc.local文件必须有执行权限：sudo chmod +x /etc/rc.local。
  • 启动命令必须在文件末尾添加，且必须加上&符号，使其在后台运行。
  • 启动命令必须在exit 0语句之前添加。
