MongoDB tar包安装

1、登录mongodb官网，下载安装包

https://www.mongodb.com/download-center/community

2.解压

tar -zxvf mongodb-linux-x86_64-amazon-3.6.9.tgz 

3.修改文件夹名称

mv mongodb-linux-x86_64-amazon-3.6.9  mongodb

4、在/usr/local/mongodb下创建文件

mkdir -p data/db

mkdir  logs

 

5、在/usr/local/mongodb/bin下新建配置

vi mongodb.conf

dbpath = /usr/local/mongodb/data/db

logpath = /usr/local/mongodb/logs/mongodb.log

port = 27017

fork = true

6、环境变量配置

vi /etc/profile

export MONGODB_HOME=/usr/local/mongodb

export PATH=$PATH:$MONGODB_HOME/bin

#保存后，重启系统配置

source /etc/profile

7、在/usr/local/mongodb/bin下

mongod -f mongodb.conf 或 ./mongod -f mongodb.conf
