解压文件

文件里没啥有用的内容，修改时间很特别到35年去了，注意到提示

 而描述的时间戳为

 再看一下文件的修改时间

 

想到的思路是，时间戳之差，根据经验猜测为ascii码，然后用python写了个脚本

#Author:Luci4n
import os,time

flag = ""
origin_time = 2000000000

for i in range(0,47):
    file = r'./{0}.txt'.format(i) # 文件路径
    get_time = int(os.path.getmtime(file)) #获取最近修改时间
    key = get_time - origin_time
    flag += chr(key)
    
print(flag)
