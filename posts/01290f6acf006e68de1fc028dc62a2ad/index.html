vue中使用animate.css

1.首先要安装animate.css

npm install animate.css --save

2.引入animate.css

import 'animate.css'

3.使用animate.css

vue组件中

<transition name="animate__animated animate__bounce"
			enter-active-class="animate__XXX"
			leave-active-class="animate__XXX">
<h2>内容文字</h2>
</transition>

其中XXX表示动画名，如swing，bounce等等
