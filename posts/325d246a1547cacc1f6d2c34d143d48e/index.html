VSCode 返回上一个光标 (上一个浏览位置) 前进/返回


更多 常用快捷键 动画演示

Ctrl + 鼠标左击跳到定义.

返回：

Windows: Alt + ← ;或者 鼠标侧键
Linux: Ctrl + Alt + - ;貌似数字键盘的减号没效果
Mac: Ctrl + -

前进：

Windows: Alt + → ;或者 鼠标侧键

查看快捷键、快捷键自定义设置：
VSCode左下角 “管理 / Manage” -> “键盘快捷方式 / Keyboard Shortcuts” -> 搜索 “前进 / Go Forward 或 返回 / Go Back”
